# 歯磨き記録アプリ（HamiLog）企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

毎日の歯磨きをタイマーで計測・記録し、習慣の質を振り返るアプリ。
歯ブラシ・歯磨き粉の使用期間も管理することで、デンタルケア全体を一元管理できる。

- 朝晩の歯磨き時間を記録して達成率を可視化
- 糸ようじの習慣化をサポート
- 歯ブラシ・歯磨き粉の交換タイミングを把握

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録によるマルチデバイス対応 |
| Phase 3（拡張） | 家族の歯磨き管理（子どもの習慣化） |

---

## 3. 主要機能

### 3.1 記録機能（MVP）
- 朝 / 夜 を選んでスタート
- 歯ブラシ：タイマーで計測 → 完了チェック
- 糸ようじ：やった / やってない チェック
- 記録は日付・時刻自動付与

### 3.2 タイマー機能
- シンプルなストップウォッチ形式
- 完了ボタンで記録に保存

### 3.3 アイテム管理機能
- **歯ブラシ登録**：名前・使い始めた日を登録
  - 経過日数を表示（例：前回交換から5日）
  - 交換時に新規登録
- **歯磨き粉登録**：名前・開封日を登録
  - 経過日数を表示（例：開封から10日）
  - 交換時に新規登録

### 3.4 振り返り機能（Phase 2）
- 朝晩の達成率（月別）
- 平均歯磨き時間
- 糸ようじのサボり率
- 歯ブラシ・歯磨き粉の交換履歴

```
│   ├── 糸ようじ達成率
│   └── アイテム交換履歴
└── 設定画面（Phase 2）
    └── プロフィール・ログイン
```
