# プロジェクト管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

複数のプロジェクトを同時進行する際、タスクの管理・進捗把握・期限管理が難しくなる。プロジェクト管理アプリは以下の課題を解決する。

- プロジェクトのタスク・マイルストーン・進捗を一元管理する
- ガントチャートによる視覚的なスケジュール管理
- 個人プロジェクトからチームプロジェクトまで対応する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身の個人プロジェクト管理 |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 複数メンバーでのチームプロジェクト管理 |

---

## 3. 主要機能

### 3.1 プロジェクト管理機能（MVP）
- プロジェクトの作成・編集・削除
- 登録項目：
  - プロジェクト名・説明
  - 開始日・終了日
  - ステータス（計画中・進行中・完了・中断）
  - カテゴリ・優先度
  - 進捗率

### 3.2 タスク管理
- タスクの作成・担当・期限設定
- サブタスクによる作業分解
- タスクの依存関係設定

### 3.3 ビュー切り替え
- カンバンボード（ToDo・進行中・完了）
- ガントチャート（スケジュール表示）
- リストビュー

### 3.4 マイルストーン管理
- 重要な節目（マイルストーン）の設定
- マイルストーンに紐付いたタスク管理
- 進捗率の自動計算

### 3.5 レポート機能（Phase 3）
- プロジェクト完了報告書の生成
- 工数・コストのサマリー
- バーンダウンチャート

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC優先、スマホ対応 |
| ガントチャート | インタラクティブなガントチャート表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | プロジェクト一覧・ガントは3秒以内 |
| ドラッグ＆ドロップ | タスク移動・スケジュール変更 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（アクティブプロジェクト・直近のタスク）
├── プロジェクト一覧画面
├── プロジェクト詳細画面（カンバン/ガント/リスト切替）
├── タスク作成・編集画面
├── マイルストーン管理画面
└── 設定画面
```
