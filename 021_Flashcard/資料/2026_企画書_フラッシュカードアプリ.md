# フラッシュカードアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

記憶の定着には繰り返し学習が最も効果的であり、フラッシュカードはその代表的な手法である。フラッシュカードアプリは以下の課題を解決する。

- 間隔反復法（SRS）を使って効率的に記憶を定着させる
- 語学・資格・専門知識など幅広い分野の学習に対応する
- 自分専用のカードデッキを作成し、いつでも復習できる

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | デッキの公開・共有機能 |

---

## 3. 主要機能

### 3.1 カード・デッキ管理機能（MVP）
- カードの作成・編集・削除
- デッキの作成・管理
- カード項目：
  - 表面（質問・単語）
  - 裏面（答え・意味）
  - 補足メモ
  - 画像添付
  - タグ

### 3.2 学習モード
- フラッシュカード形式の学習
- 正解・不正解・うろ覚えの3段階評価
- 間隔反復法（SRS）による次回復習日の自動設定

### 3.3 復習スケジュール管理
- 今日復習すべきカードの表示
- 復習予定カード数の推移
- 習熟度別のカード分類

### 3.4 統計・進捗管理
- 習熟済みカードの割合
- 日別学習枚数のグラフ
- 正答率の推移

### 3.5 デッキ共有機能（Phase 3）
- 公開デッキの検索・インポート
- 自作デッキの公開・シェア

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | スワイプ操作でのカード学習 |
| データ保存 | ユーザーごとのデータ永続化 |
| SRSアルゴリズム | SM-2アルゴリズムの実装 |
| パフォーマンス | カード表示・評価は1秒以内 |

```
├── デッキ管理画面
├── 復習スケジュール画面
├── 統計画面
└── 設定画面
```
