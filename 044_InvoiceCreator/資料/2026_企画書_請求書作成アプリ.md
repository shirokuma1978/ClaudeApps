# 請求書作成アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

フリーランス・個人事業主にとって請求書の作成・管理は必須業務だが、手動での作成は手間がかかる。請求書作成アプリは以下の課題を解決する。

- 請求書をテンプレートから素早く作成・送付する
- 入金状況を管理して未入金の案件を見逃さない
- 取引先・案件ごとの収益を把握する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（フリーランス・個人事業主） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 小規模事業者の請求管理 |

---

## 3. 主要機能

### 3.1 請求書作成機能（MVP）
- 請求書の作成・編集・削除
- 登録項目：
  - 請求書番号
  - 発行日・支払期限
  - 請求先（クライアント情報）
  - 請求項目（品目・数量・単価・金額）
  - 消費税（8%・10%・非課税）
  - 合計金額
  - 備考・振込先

### 3.2 テンプレート管理
- 請求書テンプレートのカスタマイズ
- ロゴ・会社情報の設定
- よく使う品目のプリセット

### 3.3 クライアント管理
- 取引先の登録・管理
- クライアントごとの請求履歴

### 3.4 PDF出力・送付（Phase 2）
- PDF形式での請求書出力
- メール送信機能
- 入金状況の記録・管理

### 3.5 収益レポート（Phase 3）
- 月別・クライアント別の売上集計
- 未入金残高の管理
- 確定申告用の収入データ出力

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC優先、スマホ対応 |
| PDF生成 | 高品質なPDF請求書の生成 |
| データ保存 | ユーザーごとのデータ永続化 |
| インボイス対応 | 適格請求書（インボイス）フォーマット対応 |
| 番号管理 | 請求書番号の自動採番 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今月の請求・未入金サマリー）
├── 請求書一覧画面
├── 請求書作成・編集画面
├── クライアント管理画面
├── 収益レポート画面
└── 設定画面（会社情報・テンプレート・振込先）
```
