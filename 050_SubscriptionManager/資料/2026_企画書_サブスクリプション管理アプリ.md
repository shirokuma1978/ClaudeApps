# サブスクリプション管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

サブスクリプションサービスが増え、「気づいたら使っていないサービスに毎月料金を払っていた」というケースが増えている。サブスクリプション管理アプリは以下の課題を解決する。

- 契約中の全サブスクリプションを可視化して無駄を把握する
- 更新日の事前通知で不要なサービスを解約するタイミングを逃さない
- 月間・年間の総コストを把握してサービスの取捨選択を支援する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族でのサブスク管理・費用分担 |

---

## 3. 主要機能

### 3.1 サブスク登録機能（MVP）
- サービスの登録・編集・削除
- 登録項目：
  - サービス名
  - カテゴリ（動画・音楽・クラウド・SaaS等）
  - 料金・通貨
  - 課金サイクル（月次・年次・週次）
  - 次回更新日
  - 利用頻度（高・中・低・ほぼ未使用）
  - 支払い方法
  - メモ・満足度

### 3.2 ダッシュボード
- 月額合計・年額合計の表示
- カテゴリ別の費用内訳グラフ
- 今月の更新予定一覧

### 3.3 更新アラート機能
- 更新日の7日前・3日前・前日の通知
- 解約を検討すべきサービスの提案

### 3.4 コスト分析（Phase 2）
- 利用頻度に対するコストパフォーマンス
- 不要なサービスの特定
- 節約シミュレーション

### 3.5 家族共有（Phase 3）
- 家族全員のサブスクを統合管理
- サービスの共有（ファミリープラン）管理

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| 通知 | ブラウザ通知による更新日アラート |
| 多通貨 | 円・ドル等の複数通貨対応 |
| パフォーマンス | ダッシュボード表示は2秒以内 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（月額コスト・更新予定・コスト内訳）
├── サービス一覧画面（カテゴリ別）
├── サービス登録・編集画面
├── コスト分析画面
├── 更新スケジュール画面
└── 設定画面（通知設定）
```

- Phase 1：全サブスクを登録して月間コストを把握できること

- Phase 3：家族全体のサブスクを最適化できること
