# ストレッチ・ヨガ記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

在宅ワークの普及で身体の凝り・柔軟性低下が社会問題化している。ストレッチ・ヨガ記録アプリは以下の課題を解決する。

- 毎日のストレッチ・ヨガを継続するための記録ツールを提供する
- 柔軟性の改善をトラッキングして成長を実感できる
- 部位別のストレッチを管理し、全身をバランスよくケアする

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | ヨガ仲間との練習共有 |

---

## 3. 主要機能

### 3.1 セッション記録機能（MVP）
- ストレッチ/ヨガセッションの登録
- 登録項目：
  - 日付・時間
  - セッション種類（朝ストレッチ・ヨガ・夜ストレッチ等）
  - 実施時間
  - 対象部位（首・肩・腰・足等）
  - 体の状態（ビフォー/アフター）
  - 感想・気づき

### 3.2 ポーズ・ルーティン管理
- よく使うストレッチ/ヨガのルーティン保存
- ポーズの説明・注意事項メモ
- 所要時間の設定

### 3.3 柔軟性トラッキング
- 前屈・開脚などの柔軟性指標の記録
- 数値の推移グラフ

### 3.4 連続記録・統計
- 継続日数ストリーク
- 週間・月間セッション数
- 部位別のケア頻度

### 3.5 プログラム管理（Phase 3）
- 週次ヨガプログラムの作成
- ポーズ動画のリンク管理

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | ストレッチ中のスマホ利用を考慮 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録入力は2秒以内 |
| タイマー | ポーズ保持タイマー機能 |

## 5. 画面構成（概要）

