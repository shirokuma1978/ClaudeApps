# 行ったことある場所記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

「全都道府県制覇」「日本の世界遺産全制覇」など、訪問場所の記録・達成感を楽しみたいというニーズがある。行ったことある場所記録アプリは以下の課題を解決する。

- 訪問した場所・スポットを記録して地図で可視化する
- 都道府県・国などの制覇率を管理して達成感を味わう
- まだ行けていない場所のバケットリストを管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（旅行好き・地域探索好き） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 旅仲間との訪問マップ共有 |

---

## 3. 主要機能

### 3.1 場所記録機能（MVP）
- 訪問場所の登録・編集・削除
- 登録項目：
  - 場所名
  - 都道府県・市区町村・国
  - 訪問日
  - カテゴリ（自然・観光地・グルメ・温泉等）
  - 評価
  - 感想・メモ
  - 写真

### 3.2 地図・マップビュー
- 日本地図での訪問都道府県の塗りつぶし
- 世界地図での訪問国の表示
- 都道府県内の市区町村マップ

### 3.3 制覇チャレンジ
- 都道府県制覇率の表示
- 日本の世界遺産・国宝制覇チャレンジ
- ご当地グルメ制覇チャレンジ

### 3.4 バケットリスト
- 行きたい場所のリスト
- 優先度設定
- 行った/まだの状態管理

### 3.5 旅仲間共有（Phase 3）
- 訪問マップの公開設定
- 友人の訪問場所との比較

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| マップ | SVGマップ / Google Maps API |
| データ保存 | ユーザーごとのデータ永続化 |
| 画像 | 訪問地の写真アップロード |
| パフォーマンス | マップ表示は3秒以内 |
