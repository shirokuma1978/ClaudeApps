# 旅行予算管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

旅行中の費用管理は楽しさを損なわず適切に行うことが重要である。旅行予算管理アプリは以下の課題を解決する。

- 旅行の予算設定と実際の支出をリアルタイムで管理する
- カテゴリ別（交通・宿泊・食事・観光等）の費用を把握する
- 海外旅行での外貨換算も含めた費用管理をする

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人旅行者） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | グループ旅行での費用分担管理 |

---

## 3. 主要機能

### 3.1 旅行予算設定（MVP）
- 旅行の登録と予算設定
- カテゴリ別予算の設定：
  - 交通費（飛行機・電車・バス等）
  - 宿泊費
  - 食費
  - 観光・入場料
  - お土産・ショッピング
  - その他

### 3.2 費用入力・管理
- 旅行中の支出をリアルタイムに入力
- 残予算のリアルタイム表示
- カテゴリ別の進捗バー

### 3.3 外貨換算機能
- 複数通貨での費用入力
- 為替レートの設定（出発前・現地実績）
- 円換算での合計表示

### 3.4 費用分析（Phase 2）
- カテゴリ別費用の円グラフ
- 日別の支出推移
- 予算オーバー警告

### 3.5 グループ精算（Phase 3）
- 立替費用の記録
- グループメンバーとの割り勘計算
- 精算額の自動計算

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ優先、PCブラウザ対応 |
| 外貨対応 | 主要通貨の換算対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 費用入力は即時 |
| オフライン | 旅行中のオフライン入力対応（将来） |

```

├── ホーム（進行中の旅行・残予算）

├── 予算状況ダッシュボード
├── 費用分析画面

```
