# スキルマップアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

自分のスキルセットを客観的に把握し、強み・弱みを可視化することはキャリア形成に重要である。スキルマップアプリは以下の課題を解決する。

- 自分のスキル・知識を体系的に整理・可視化する
- スキルギャップを把握して学習計画に反映させる
- 転職・評価・自己PRの材料としてスキルを整理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | チームのスキルマップ管理 |

---

## 3. 主要機能

### 3.1 スキル登録機能（MVP）
- スキルの登録・編集・削除
- 登録項目：
  - スキル名
  - カテゴリ（技術・ビジネス・語学・資格等）
  - 習熟度（1〜5段階）
  - 年数・経験
  - 具体的な実績・プロジェクト
  - 現在学習中かどうか
  - 目標習熟度

### 3.2 スキルマップ可視化
- レーダーチャートでのスキル全体像
- カテゴリ別のスキルツリー表示
- 強みスキル・改善が必要なスキルの色分け

### 3.3 ギャップ分析
- 目標習熟度と現在のギャップ表示
- 補うべきスキルの優先度付け
- 学習計画への反映

### 3.4 スキル証明管理（Phase 2）
- 資格・認定証の管理
- プロジェクト実績との紐付け
- 習熟度の更新履歴

### 3.5 チームスキルマップ（Phase 3）
- チームメンバーのスキルの一覧
- スキルギャップの特定
- 人材配置の最適化

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| グラフ | レーダーチャート・ツリー表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| エクスポート | PDF形式でのスキルシート出力 |
| パフォーマンス | グラフ表示は3秒以内 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（スキルマップ概要・最近の更新）
├── スキル一覧・登録画面
├── スキルマップ可視化画面
├── ギャップ分析画面
├── 資格・証明管理画面
└── 設定画面
```
