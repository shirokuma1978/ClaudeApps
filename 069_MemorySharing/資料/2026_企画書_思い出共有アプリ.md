# 思い出共有アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

特別な体験・思い出を大切な人と共有したいというニーズは普遍的である。思い出共有アプリは以下の課題を解決する。

- 友人・家族との思い出を記録・共有するプライベートな空間を提供する
- SNSとは違う、特定の人だけと共有できる思い出アルバムを作る
- 過去の思い出をリマインドして関係を深める

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用・思い出記録） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 友人・カップル・家族との思い出共有 |

---

## 3. 主要機能

### 3.1 思い出記録機能（MVP）
- 思い出の記録・編集・削除
- 登録項目：
  - タイトル
  - 日付・場所
  - 関係した人
  - 写真・動画
  - 本文（日記形式）
  - タグ・感情
  - 公開範囲（自分・特定の人・グループ）

### 3.2 思い出アルバム
- 時系列の思い出タイムライン
- 人物別・場所別・タグ別のアルバム
- 「〇年前の今日」のリマインド機能

### 3.3 共有・コラボレーション
- 特定の人との思い出の共有
- 相手からのコメント・リアクション
- 共同で思い出を作成・編集

### 3.4 プライバシー管理
- 細かい公開範囲の設定
- 特定の人への非公開設定

### 3.5 思い出振り返り（Phase 3）
- 年間ハイライトの自動生成
- 記念日・節目の思い出まとめ

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 画像 | 高品質写真のアップロード・表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| セキュリティ | プライバシー保護・暗号化 |
| パフォーマンス | 写真一覧は3秒以内 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近の思い出・今日の記念日）
├── 思い出タイムライン
├── 思い出作成・編集画面
├── 思い出詳細画面
├── アルバム・コレクション画面
└── 設定画面（プライバシー設定）
```
