# 感謝日記アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

感謝の気持ちを日記として記録する「グラティチュードジャーナル」は、ポジティブ心理学において幸福感向上に効果的と証明されている。感謝日記アプリは以下の課題を解決する。

- 毎日感謝できることを3つ記録する習慣を形成する
- ネガティブな思考パターンを変えて人生の満足度を高める
- 感謝の記録を振り返ることで過去の出来事を再認識する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・パートナーへの感謝を伝えるツール |

---

## 3. 主要機能

### 3.1 感謝記録機能（MVP）
- 毎日の感謝記録（1日3つ推奨）
- 記録項目：
  - 今日感謝できること（1〜5つ）
  - 感謝の対象（人・出来事・モノ）
  - 詳細メモ
  - 感情タグ（嬉しい・ありがたい・感動等）

### 3.2 ストリーク管理
- 連続記録日数の表示
- 継続モチベーションの維持

### 3.3 振り返り機能
- 「1年前の今日」の感謝の振り返り
- 特定の人への感謝の統計
- 最も感謝しているカテゴリの分析

### 3.4 幸福度グラフ（Phase 2）
- 毎日の幸福度スコアと感謝記録の相関
- 感謝の対象別の頻度分析

### 3.5 感謝メッセージ送信（Phase 3）
- 感謝した人へのメッセージ送信機能
- 匿名での感謝ルーム
