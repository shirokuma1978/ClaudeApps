# ブログ下書き管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

ブロガーにとって、ネタのストック・下書き管理・公開スケジュール管理が課題である。ブログ下書き管理アプリは以下の課題を解決する。

- ブログネタのアイデアを素早くメモし、下書きに発展させる
- 複数ブログ・複数媒体の記事を一元管理する
- 公開スケジュールを管理して継続的な発信を支援する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人ブロガー） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 複数ライターでのチームブログ管理 |

---

## 3. 主要機能

### 3.1 記事管理機能（MVP）
- 記事の作成・編集・削除
- 記事項目：
  - タイトル
  - カテゴリ・タグ
  - ステータス（アイデア・下書き・執筆中・レビュー待ち・公開済み）
  - 対象ブログ・媒体
  - 公開予定日
  - 文字数カウント
  - 本文（マークダウン対応）

### 3.2 ネタ帳機能
- ブログネタのストック
- ネタの優先度・難易度の設定
- ネタから記事への変換

### 3.3 公開スケジュール管理
- カレンダー形式の公開スケジュール表示
- 月間・週間の投稿頻度の目標設定
- 未公開記事の催促リマインダー

### 3.4 統計・分析（Phase 2）
- 月別投稿数の推移
- カテゴリ別の記事数
- 書きかけ記事の放置期間アラート

### 3.5 チームブログ機能（Phase 3）
- 複数ライターへの記事アサイン
- 編集者によるレビュー・フィードバック
- 承認ワークフロー

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC優先、スマホ対応 |
| エディタ | マークダウン対応エディタ（プレビュー付き） |
| データ保存 | ユーザーごとのデータ永続化 |
| 検索 | 記事全文検索 |
| パフォーマンス | 記事一覧表示は3秒以内 |

```
├── ログイン画面

├── ホーム（執筆中の記事・公開予定）
├── 記事一覧（ステータス別）
├── 記事エディタ画面
├── ネタ帳画面
├── 公開スケジュールカレンダー
├── 統計画面
└── 設定画面（ブログ・カテゴリ設定）
```
