# 面接準備管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

転職・就活の面接では、企業研究・想定質問への回答・自己PRの準備が不可欠である。面接準備管理アプリは以下の課題を解決する。

- 複数の選考企業と面接の準備状況を一元管理する
- 想定質問とその回答を記録・練習する
- 面接後のフィードバックを蓄積して次に活かす

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（転職活動中・就活生） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | キャリアコーチとの準備状況共有 |

---

## 3. 主要機能

### 3.1 企業・選考管理（MVP）
- 応募企業の登録・管理
- 登録項目：
  - 会社名・業種・規模
  - 応募ポジション
  - 選考ステップ（書類・1次・2次・最終）
  - 選考日程
  - ステータス（選考中・内定・不合格・辞退）
  - 企業メモ・会社情報

### 3.2 想定質問・回答管理
- 頻出質問の登録
- 回答の作成・練習
- 良い回答・改善点のメモ
- STAR法（状況・課題・行動・結果）でのフォーマット

### 3.3 自己PR・志望動機管理
- 会社別の自己PR・志望動機の管理
- 強み・経験のエピソードライブラリ
- 使い回せる回答テンプレート

### 3.4 面接後フィードバック記録（Phase 2）
- 面接での質問記録
- 実際に使った回答・感触
- 次回への改善点

### 3.5 キャリアコーチ連携（Phase 3）
- コーチへの準備状況の共有
- フィードバックのやり取り

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| セキュリティ | 個人情報・転職活動の適切な保護 |
| 検索 | 企業名・ポジション・質問での検索 |

```
├── 想定質問・回答管理画面

```
