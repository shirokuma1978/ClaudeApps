# 日記・ジャーナルアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

毎日の出来事や感情を記録することは、自己成長・メンタルヘルス維持に非常に効果的である。日記・ジャーナルアプリは以下の課題を解決する。

- 思考の整理と感情のアウトプットで精神的な健康を維持する
- 過去の記録を振り返ることで、成長を実感できる
- プライベートな内容を安全に保存・管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・パートナーとの共有日記機能 |

---

## 3. 主要機能

### 3.1 日記記録機能（MVP）
- 日記の作成・編集・削除
- 記録項目：
  - 日付
  - タイトル
  - 本文（マークダウン対応）
  - 気分（5段階）
  - タグ
  - 天気（自動取得または手動入力）

### 3.2 テンプレート機能
- ジャーナリングテンプレート（感謝3つ・今日の学び・明日の目標など）
- カスタムテンプレートの作成

### 3.3 検索・振り返り機能
- キーワード検索
- タグ・日付・気分でのフィルタリング
- 「今日の1年前」「今日の1ヶ月前」の自動表示

### 3.4 統計機能
- 気分の推移グラフ
- 記録頻度カレンダー
- よく使うタグの集計

### 3.5 エクスポート機能（Phase 3）
- PDF・テキスト形式での書き出し
- 年間アーカイブの作成

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| セキュリティ | プライベートデータの暗号化保存 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録・閲覧は3秒以内 |
| バックアップ | 定期的な自動バックアップ |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近の日記一覧・今日の記録催促）
├── 日記作成画面
├── 日記詳細・編集画面
├── 検索・フィルター画面
├── カレンダービュー
├── 統計・気分グラフ画面
└── 設定画面（テンプレート管理・プロフィール）
```
