# アニメ視聴記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

アニメは毎クール多数の新作が放送される中、視聴管理・追いかけるべき作品の整理が課題である。アニメ視聴記録アプリは以下の課題を解決する。

- 視聴済み・視聴中・視聴予定のアニメを一元管理する
- 感想・評価を記録してアニメ歴を積み上げる
- 今期放送中のアニメの視聴進捗を管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（アニメファン） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | アニメ好き仲間との視聴記録共有 |

---

## 3. 主要機能

### 3.1 アニメ記録機能（MVP）
- アニメの登録・編集・削除
- 登録項目：
  - タイトル
  - 放送年・クール・スタジオ
  - ジャンル
  - 視聴ステータス（視聴済み・視聴中・視聴予定・中断）
  - 評価（10点満点）
  - 視聴完了話数（連続ものの進捗）
  - 感想・レビュー
  - お気に入りキャラクター

### 3.2 アニメ情報取得
- 外部APIを使ったアニメ情報の自動取得
- キービジュアル・スタッフ情報

### 3.3 視聴状況ダッシュボード
- 今期アニメの視聴進捗一覧
- 視聴済みアニメ数カウント
- 総視聴時間の計算

### 3.4 統計・可視化
- ジャンル別・年代別の視聴分布
- 平均評価スコアの推移
- 好きなスタジオの分析

### 3.5 おすすめ・共有（Phase 3）
- 友人へのおすすめ機能
- レビューの共有
- 今期アニメの注目情報共有

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 外部API | AniList API または MyAnimeList API |
| データ保存 | ユーザーごとのデータ永続化 |
| 画像 | アニメビジュアル画像の表示 |
| パフォーマンス | 一覧表示は3秒以内 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今期の視聴状況・最近記録）
├── 視聴記録一覧（ステータス別）
├── アニメ検索・登録画面
├── 詳細・感想記録画面
├── 統計・グラフ画面
└── 設定画面
```
