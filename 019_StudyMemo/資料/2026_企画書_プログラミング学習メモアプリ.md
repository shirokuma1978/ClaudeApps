# プログラミング学習メモアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

プログラミング学習では、覚えるべきコード・概念・エラー解決方法が大量に発生する。プログラミング学習メモアプリは以下の課題を解決する。

- 学習中のコードスニペット・概念を構造化して保存する
- エラーとその解決策を記録して同じミスを繰り返さない
- 学んだ内容を後から検索・参照できる個人ナレッジベースを構築する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | プログラミング学習コミュニティでのメモ共有 |

---

## 3. 主要機能

### 3.1 メモ登録機能（MVP）
- メモの作成・編集・削除
- 登録項目：
  - タイトル
  - カテゴリ（言語・フレームワーク・概念など）
  - 本文（マークダウン対応）
  - コードブロック（シンタックスハイライト）
  - タグ
  - 参照URL・参考資料

### 3.2 コードスニペット管理
- 言語別のコードスニペット保存
- コードのコピーボタン
- シンタックスハイライト（Python・JavaScript・SQL等）

### 3.3 エラーログ機能
- エラーメッセージ・原因・解決策の記録
- エラーの種類・発生状況のタグ付け

### 3.4 検索・整理機能
- 全文検索
- カテゴリ・タグでのフィルタリング
- 最近見たメモ・よく参照するメモ

### 3.5 学習ロードマップ連携（Phase 3）
- スキルマップとの紐付け
- 学習進捗の可視化

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC優先、スマホ対応 |
| エディタ | マークダウン対応エディタ（プレビュー付き） |
| シンタックスハイライト | 主要プログラミング言語対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| 検索 | 全文検索（コード含む） |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近のメモ・よく使うメモ）
├── メモ一覧画面（カテゴリ別サイドバー）
├── メモ作成・編集画面（マークダウンエディタ）
├── メモ詳細画面
├── コードスニペット一覧画面
├── エラーログ画面
└── 設定画面（カテゴリ・タグ管理）
```

## 8. 成功指標

- Phase 1：プログラミング学習中のメモを100件以上登録できること

- Phase 3：同じエラーで詰まる時間がゼロになること
