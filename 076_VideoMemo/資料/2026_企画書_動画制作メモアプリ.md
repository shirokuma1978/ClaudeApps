# 動画制作メモアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

YouTubeやTikTok等の動画クリエイターが増える中、動画制作のアイデア・企画・進捗管理が課題となっている。動画制作メモアプリは以下の課題を解決する。

- 動画のアイデア・企画をストックして制作に活かす
- 撮影・編集・公開のワークフローを管理する
- 過去の動画の反応・改善点を記録して次回に活かす

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人YouTuber・動画クリエイター） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 動画制作チームでのプロジェクト管理 |

---

## 3. 主要機能

### 3.1 動画企画管理（MVP）
- 動画企画の登録・編集・削除
- 登録項目：
  - タイトル・概要
  - ジャンル・カテゴリ
  - ターゲット視聴者
  - 台本・シナリオ
  - 撮影場所・機材メモ
  - ステータス（アイデア・台本作成中・撮影予定・編集中・公開済み）
  - 公開予定日

### 3.2 台本・シナリオ管理
- マークダウン対応のエディタ
- シーン分け・カット割りのメモ
- ナレーション・テロップの管理

### 3.3 制作進捗管理
- 撮影→編集→サムネイル→公開のステップ管理
- 制作時間の記録
- 公開スケジュールのカレンダー表示

### 3.4 動画実績記録（Phase 2）
- 公開後の視聴数・いいね数の記録
- 高パフォーマンス動画の分析
- 改善メモの記録

### 3.5 チーム制作（Phase 3）
- 役割分担（撮影・編集・サムネイル）
- 共同台本編集

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| エディタ | マークダウン対応台本エディタ |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 一覧表示は3秒以内 |
| 検索 | タイトル・タグ・ステータスでの検索 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（制作中の動画・公開スケジュール）
├── 動画企画一覧（ステータス別）
├── 動画企画作成・編集画面
├── 台本エディタ
├── 公開スケジュールカレンダー
├── 実績・分析画面
└── 設定画面
```
