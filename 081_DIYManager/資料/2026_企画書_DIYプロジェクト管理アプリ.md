# DIYプロジェクト管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

DIY・日曜大工に取り組む人が増える中、材料管理・設計メモ・進捗管理が課題となっている。DIYプロジェクト管理アプリは以下の課題を解決する。

- DIYプロジェクトの設計・材料・工程を管理する
- 材料費・工具費の記録で総コストを把握する
- 完成作品を記録してノウハウを蓄積する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（DIY愛好家） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | DIY仲間とのノウハウ共有 |

---

## 3. 主要機能

### 3.1 プロジェクト管理機能（MVP）
- プロジェクトの登録・編集・削除
- 登録項目：
  - プロジェクト名
  - カテゴリ（家具・インテリア・修繕等）
  - 設計メモ・図面（手書き写真）
  - 難易度・所要時間
  - ステータス（計画中・進行中・完成）
  - 完成写真

### 3.2 材料管理
- 必要材料・工具リスト
- 購入済み/未購入の管理
- 購入先・金額の記録
- 余った材料の在庫管理

### 3.3 工程管理
- 作業ステップの登録
- 各工程の進捗チェック
- 困ったことのメモ

### 3.4 コスト管理（Phase 2）
- 材料費・工具費の集計
- プロ依頼との比較コスト
- プロジェクト別の費用履歴

### 3.5 DIY記録ギャラリー（Phase 3）
- ビフォーアフター写真
- 完成作品の公開共有

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 画像 | 作業過程・完成写真のアップロード |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | プロジェクト一覧は3秒以内 |
| 計算 | 材料費の自動集計 |
