# 創作小説メモアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

小説・物語の創作では、キャラクター設定・世界観・プロット管理が複雑になる。創作小説メモアプリは以下の課題を解決する。

- キャラクター・世界観・プロットを一元管理して創作の一貫性を保つ
- 複数作品を並行して管理する
- 章・シーンの進捗を管理して完成に向けてモチベーションを維持する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人創作者・小説家志望） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 合作・創作サークルでの共同管理 |

---

## 3. 主要機能

### 3.1 作品管理機能（MVP）
- 作品の登録・編集・削除
- 登録項目：
  - タイトル・あらすじ
  - ジャンル（ファンタジー・SF・恋愛等）
  - ステータス（企画中・執筆中・完成・投稿済み）
  - 字数カウント（目標字数・進捗）
  - 執筆開始日・公開予定日

### 3.2 キャラクター管理
- キャラクターの詳細プロフィール
- 外見・性格・過去・能力
- キャラクター間の関係図

### 3.3 世界観・設定管理
- 世界観・魔法・ルール等の設定メモ
- 地図・場所の記録
- 用語辞典

### 3.4 プロット・シーン管理
- 章・シーン単位の構成管理
- 各シーンのメモ・テーマ
- 時系列の整理

### 3.5 執筆支援（Phase 3）
- 本文エディタとの統合
- 文字数・ペースのトラッキング

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC優先、スマホ対応 |
| エディタ | 書きやすいテキストエディタ |
| データ保存 | ユーザーごとのデータ永続化 |
| 検索 | 設定・キャラクターの全文検索 |
| パフォーマンス | 一覧表示は3秒以内 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（執筆中の作品・今日の進捗）
├── 作品一覧画面
├── 作品詳細ページ
│   ├── キャラクター管理
│   ├── 世界観設定管理
│   └── プロット・構成管理
├── 全体検索画面
└── 設定画面
```
