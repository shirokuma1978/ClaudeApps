# 音楽制作メモアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

音楽制作者・作曲家が曲のアイデア・コード進行・制作メモを管理するための専用ツールが求められている。音楽制作メモアプリは以下の課題を解決する。

- 楽曲アイデア・メロディのスケッチを素早く記録する
- 制作中の楽曲の進捗・バージョン管理をする
- コード進行・歌詞・構成メモを整理して制作効率を高める

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（アマチュア・プロ音楽制作者） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | バンド・制作チームでの楽曲共同管理 |

---

## 3. 主要機能

### 3.1 楽曲メモ管理（MVP）
- 楽曲の登録・編集・削除
- 登録項目：
  - 曲名・仮タイトル
  - ジャンル・テンポ・キー
  - ステータス（アイデア・制作中・完成・ボツ）
  - コード進行メモ
  - 歌詞メモ
  - 曲構成（イントロ・Aメロ・Bメロ・サビ等）
  - 使用機材・プラグイン
  - 音声メモのURL・リンク

### 3.2 コード進行ライブラリ
- よく使うコード進行のストック
- コード進行の名前・ムード分類
- 曲への紐付け

### 3.3 制作進捗管理
- 制作ステップの管理（デモ・リファレンス・本録音・ミックス等）
- タスクリスト
- 制作時間の記録

### 3.4 インスピレーション管理（Phase 2）
- 参考にしたい曲・アーティストのメモ
- 感情・シーン別の楽曲アイデア分類

### 3.5 バンド・チーム共有（Phase 3）
- メンバーへのデモ共有
- フィードバック・コメント機能

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| テキスト | コード進行のテキスト表記 |
| データ保存 | ユーザーごとのデータ永続化 |
| 音声 | 音声メモのURL管理 |
| 検索 | ジャンル・キー・ステータスでの検索 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（制作中の楽曲・最近のメモ）
├── 楽曲一覧画面
├── 楽曲詳細・編集画面
├── コード進行ライブラリ画面
├── 制作進捗管理画面
└── 設定画面
```
