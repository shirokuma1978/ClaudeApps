# 引っ越し準備チェックリストアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

引っ越しは人生で最も手続きが多いイベントの一つであり、やることの多さに圧倒されることがある。引っ越し準備チェックリストアプリは以下の課題を解決する。

- 引っ越しに必要な全ての作業をリスト化して管理する
- 引っ越し日からの逆算スケジュールを自動生成する
- やり忘れをなくして、スムーズな引っ越しを実現する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・カップルでの引っ越し共同管理 |

---

## 3. 主要機能

### 3.1 チェックリスト機能（MVP）
- プリセットの引っ越しチェックリスト
- カスタムタスクの追加
- チェックオフ機能
- カテゴリ別整理：
  - 旧居退去手続き
  - 新居入居手続き
  - 各種住所変更
  - 引っ越し業者手続き
  - ライフライン手続き

### 3.2 タイムライン管理
- 引っ越し日の設定
- 「引っ越しXX日前」でのタスク自動スケジュール
- 期限付きリマインダー

### 3.3 手続き情報管理
- 各手続きの方法メモ
- 必要書類リスト
- 連絡先・受付時間のメモ

### 3.4 費用管理（Phase 2）
- 引っ越し関連費用の記録
- 見積もり比較
- 引っ越し予算の管理

### 3.5 家族共有（Phase 3）
- パートナー・家族との共同タスク管理
- 担当分けと進捗共有

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| テンプレート | 状況別プリセット（一人暮らし・家族等） |
| データ保存 | ユーザーごとのデータ永続化 |
| 通知 | 期限前のタスクリマインダー |
| パフォーマンス | チェック操作は即時 |

## 5. 画面構成（概要）

```

```
