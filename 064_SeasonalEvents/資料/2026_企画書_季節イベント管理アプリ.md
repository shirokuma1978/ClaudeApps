# 季節イベント管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

クリスマス・年賀状・大掃除など季節のイベントは毎年発生するが、「もうそんな時期か」と慌てることが多い。季節イベント管理アプリは以下の課題を解決する。

- 年間の季節イベント・行事を一覧管理して事前準備を促す
- 準備タスクを逆算スケジュールで自動生成する
- 毎年の準備記録を残して翌年の参考にする

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族でのイベント管理・準備分担 |

---

## 3. 主要機能

### 3.1 イベント管理機能（MVP）
- 季節イベントの登録・管理
- プリセットイベント：
  - 年賀状・お正月準備
  - バレンタイン・ホワイトデー
  - お花見・ゴールデンウィーク
  - 夏休み・お盆
  - ハロウィン・クリスマス
  - 大掃除
- カスタムイベントの追加

### 3.2 準備タスク管理
- イベントごとの準備タスクリスト
- 逆算スケジュールによるタスク期限の設定
- チェックオフ機能

### 3.3 年間カレンダービュー
- 年間のイベント一覧表示
- 月別の準備タスク表示
- 直近イベントのカウントダウン

### 3.4 過去記録の活用
- 昨年の準備記録の参照
- よかった点・改善点のメモ
- 使ったアイテム・購入先の記録

### 3.5 家族共有（Phase 3）
- 家族間での準備タスク分担
- プレゼント計画の共有

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| カレンダー | 年間カレンダービュー |

| パフォーマンス | カレンダー表示は3秒以内 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（直近イベント・今月のタスク）
├── 年間カレンダー画面

├── 過去記録参照画面
└── 設定画面（イベント設定・通知設定）
```
