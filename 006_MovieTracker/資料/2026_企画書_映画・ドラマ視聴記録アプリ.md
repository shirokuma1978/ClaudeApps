# 映画・ドラマ視聴記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

映画・ドラマは多くの人の娯楽であるが、「あの映画もう見たっけ？」「どこまで見たっけ？」という管理の煩雑さがある。視聴記録アプリは以下の課題を解決する。

- 視聴済み・視聴中・ウォッチリストを一元管理する
- 感想・評価を記録し、友人への推薦に役立てる
- どのジャンルをよく見るか、視聴傾向を把握する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 友人との視聴リスト共有・おすすめ交換 |

---

## 3. 主要機能

### 3.1 視聴記録機能（MVP）
- 映画・ドラマの登録・編集・削除
- 登録項目：
  - タイトル・監督・出演者
  - ジャンル
  - 視聴ステータス（見たい・視聴中・視聴済み）
  - 評価（5段階）
  - 視聴日
  - 感想・レビュー
  - ドラマのシーズン・エピソード進捗

### 3.2 作品検索・情報取得
- TMDb API を使った映画・ドラマ情報の自動取得
- ポスター画像・あらすじ・キャスト情報

### 3.3 ウォッチリスト管理
- 見たい作品の登録
- 優先度設定
- ストリーミングサービス別の管理

### 3.4 統計・可視化機能
- 年間視聴本数
- ジャンル別・国別の内訳
- お気に入り監督・俳優の分析

### 3.5 共有機能（Phase 3）
- レビューの共有
- 友人へのおすすめ送信
- 共同ウォッチリスト

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 外部API | TMDb（The Movie Database）API |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 検索・情報取得は5秒以内 |
| 画像 | ポスター画像のキャッシュ対応 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近の記録・続きを見る作品）
├── 記録一覧（ステータス別タブ）
├── 作品検索・登録画面
├── 作品詳細画面
├── ウォッチリスト画面
├── 統計・グラフ画面
└── 設定画面
```
