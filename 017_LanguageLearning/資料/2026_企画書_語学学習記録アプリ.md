# 語学学習記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

語学学習は長期にわたる取り組みであり、継続的な記録と振り返りが上達の鍵となる。語学学習記録アプリは以下の課題を解決する。

- 毎日の学習時間・内容を記録して習慣化を促進する
- 語彙・フレーズのメモを蓄積して効率的に復習できる
- 学習の進捗を可視化してモチベーションを維持する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 語学学習仲間との進捗共有 |

---

## 3. 主要機能

### 3.1 学習記録機能（MVP）
- 学習セッションの登録
- 記録項目：
  - 学習日・時間帯
  - 学習言語
  - 学習時間
  - 学習内容（テキスト・アプリ・ドラマ視聴など）
  - 今日の気づき・メモ
  - 習熟度（主観評価）

### 3.2 語彙・フレーズノート
- 新しく覚えた単語・フレーズの登録
- 意味・例文・発音メモ
- 復習ステータス管理

### 3.3 学習時間トラッキング
- タイマー機能（学習中の計測）
- 月別・週別の学習時間グラフ
- 目標時間との比較
- 学習連続日数（ストリーク）

### 3.4 CEFR目標管理（Phase 2）
- CEFR（A1〜C2）目標設定
- 各スキル（読・書・聞・話）の自己評価
- 資格試験の目標設定と進捗管理

### 3.5 コンテンツ管理（Phase 3）
- 学習に使ったリソースの記録
- YouTubeチャンネル・ポッドキャスト・アプリのお気に入り管理

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | 学習中のスマホ入力を考慮 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録入力は1秒以内 |
| 多言語 | 複数言語の学習を並行管理 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今日の学習状況・連続記録）
├── 学習記録入力画面（タイマー付き）
├── 語彙・フレーズノート画面
├── 学習統計・グラフ画面
├── 目標管理画面
├── 学習リソース管理画面
└── 設定画面（言語・目標設定）
```
