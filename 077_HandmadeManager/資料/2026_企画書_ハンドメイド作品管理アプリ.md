# ハンドメイド作品管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

編み物・アクセサリー・手芸などのハンドメイド作家が、作品・材料・販売を管理するための専用ツールが求められている。ハンドメイド作品管理アプリは以下の課題を解決する。

- 制作した作品の記録・在庫管理をする
- 材料費・制作時間から適切な販売価格を設定する
- 作品のポートフォリオとして記録・活用する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（ハンドメイド趣味・作家） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | ハンドメイド作家仲間との情報交換 |

---

## 3. 主要機能

### 3.1 作品管理機能（MVP）
- 作品の登録・編集・削除
- 登録項目：
  - 作品名
  - カテゴリ（アクセサリー・編み物・刺繍等）
  - 制作期間
  - 材料費
  - 制作時間
  - 販売価格・原価率
  - 在庫数
  - 写真
  - 制作メモ・パターン

### 3.2 材料・在庫管理
- 材料の在庫管理
- 使用した材料の記録
- 補充が必要な材料のリスト

### 3.3 価格計算機能
- 材料費＋時間給×工数での原価計算
- 利益率設定での販売価格提案

### 3.4 販売記録（Phase 2）
- 販売日・販売先・販売金額の記録
- 累計売上の集計
- よく売れる作品の分析

### 3.5 作品ポートフォリオ（Phase 3）
- 作品ギャラリーの公開設定
- minne・Creemaへの掲載情報管理

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 画像 | 作品写真の複数アップロード |
| データ保存 | ユーザーごとのデータ永続化 |
| 計算 | 原価・販売価格の自動計算 |
| パフォーマンス | ギャラリー表示は3秒以内 |

```

├── 価格計算ツール
├── 販売記録・収益画面
└── 設定画面
```
