# クライアント管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

フリーランス・個人事業主が複数のクライアントと取引する際、関係・案件・コミュニケーション履歴の管理が重要である。クライアント管理アプリは以下の課題を解決する。

- クライアントの基本情報・担当者・案件履歴を一元管理する
- 各クライアントとのコミュニケーション履歴を記録する
- 次のアクション・フォローアップを管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（フリーランス・個人事業主） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 小規模チームでのCRM活用 |

---

## 3. 主要機能

### 3.1 クライアント登録機能（MVP）
- クライアントの登録・編集・削除
- 登録項目：
  - 会社名
  - 担当者名・連絡先
  - 業種・規模
  - 取引開始日
  - 取引ステータス（見込み・交渉中・取引中・休眠）
  - 契約種別（単発・継続）
  - メモ・特記事項

### 3.2 案件管理
- クライアントごとの案件一覧
- 案件ステータス・金額の管理
- 見積もり・請求書との紐付け

### 3.3 コミュニケーション履歴
- メール・電話・ミーティングの記録
- 重要なやり取りのピン留め
- 次回アクションの設定

### 3.4 売上・収益管理
- クライアント別の売上集計
- 取引頻度・継続率の分析

### 3.5 フォローアップ管理（Phase 3）
- 定期コンタクトのリマインダー
- 休眠クライアントの再アプローチ提案

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| 検索 | クライアント名・担当者・業種での検索 |
| データ保存 | ユーザーごとのデータ永続化 |
| セキュリティ | 取引情報の適切な保護 |
| パフォーマンス | 一覧表示は3秒以内 |
