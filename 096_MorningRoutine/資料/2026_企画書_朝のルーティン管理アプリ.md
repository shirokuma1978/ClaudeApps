# 朝のルーティン管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

生産性の高い一日を送るためには、朝のルーティンが重要である。しかし、理想のルーティンを設計・継続することは難しい。朝のルーティン管理アプリは以下の課題を解決する。

- 毎朝のルーティンを設計して習慣化する
- 各タスクの所要時間を管理して起床から出発まで最適化する
- 継続率を可視化してモチベーションを維持する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（朝活・生産性向上を目指す人） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・パートナーとのルーティン共有 |

---

## 3. 主要機能

### 3.1 ルーティン設計機能（MVP）
- 朝のルーティン項目の登録・編集・削除
- 登録項目：
  - タスク名
  - 所要時間（分）
  - 順序
  - カテゴリ（健康・学習・準備・食事等）
  - 必須/オプション
  - メモ・ポイント
- タスクのドラッグ＆ドロップ並び替え
- 目標起床時刻と逆算スケジュール

### 3.2 タイマー機能
- ルーティン開始ボタンで順番通りに進行
- 各タスクのカウントダウンタイマー
- 次のタスクへの自動移行
- 全体の進捗表示

### 3.3 実績記録・継続管理
- 毎朝の完了記録
- 連続達成日数（ストリーク）
- 完了率の週次・月次グラフ
- 各タスクの平均所要時間の蓄積

### 3.4 ルーティン最適化（Phase 2）
- 実績データを基にした所要時間の自動調整
- 曜日別ルーティンの設定
- 完了率が低いタスクの見直し提案

### 3.5 家族共有（Phase 3）
- 家族のルーティンの共有
- 「今日のルーティン完了」の報告機能
- 家族での朝活チャレンジ

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| タイマー | バックグラウンド動作対応 |
| 通知 | ルーティン開始・タスク切り替えの通知 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | タイマー起動は1秒以内 |
