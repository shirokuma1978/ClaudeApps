# コレクション管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

フィギュア・カード・切手・コインなど多様なコレクターが、コレクションの管理に困っている。コレクション管理アプリは以下の課題を解決する。

- コレクションの現物・デジタルを一元管理し、何を持っているか把握する
- 価値・購入価格・現在価格を管理して資産としても把握する
- 未所持アイテムのウィッシュリストで収集を計画的に進める

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | コレクター仲間との交換・売買サポート |

---

## 3. 主要機能

### 3.1 コレクションアイテム登録（MVP）
- アイテムの登録・編集・削除
- 登録項目：
  - アイテム名
  - コレクションカテゴリ（フィギュア・カード等）
  - シリーズ・セット名
  - 状態（未開封・開封済み・使用中）
  - 購入日・購入価格
  - 現在の推定価格
  - 写真
  - 保管場所
  - メモ

### 3.2 コレクションセット管理
- シリーズ/セット単位での管理
- 収集率（何%コンプリートか）の表示
- 未所持アイテムの一覧

### 3.3 ウィッシュリスト
- 欲しいアイテムの登録
- 価格帯・優先度設定

### 3.4 価値・資産管理
- 購入総額の計算
- 推定現在価値の集計
- プロフィット/ロスの計算

### 3.5 コレクター間交流（Phase 3）
- コレクションの公開展示
- 交換・売買の申し出機能

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| 画像 | 複数写真のアップロード |
| データ保存 | ユーザーごとのデータ永続化 |
| 汎用性 | どんなカテゴリのコレクションにも対応 |
| パフォーマンス | アイテム一覧表示は3秒以内 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（コレクションサマリー・最近の追加）
├── カテゴリ別コレクション一覧
├── アイテム登録・編集画面
├── アイテム詳細画面
├── セット管理・収集率画面
├── 資産サマリー画面
└── 設定画面
```
