# 友達の誕生日管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

友人・家族・同僚の誕生日を覚えておくことは人間関係を深める上で大切だが、連絡先が多くなると管理が難しい。友達の誕生日管理アプリは以下の課題を解決する。

- 重要な人の誕生日を一元管理して事前通知で祝い忘れを防ぐ
- プレゼント・メッセージのアイデアを記録して毎年向上させる
- 誕生日のお祝いの記録を蓄積して関係性を深める

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | グループでの誕生日お祝い計画 |

---

## 3. 主要機能

### 3.1 誕生日登録機能（MVP）
- 人物の登録・編集・削除
- 登録項目：
  - 名前
  - 誕生日（年不明でも可）
  - 関係（家族・友人・職場・恋人等）
  - 連絡先（電話・SNS）
  - 好み・嗜好メモ
  - プレゼント履歴
  - 誕生日アラート（何日前に通知）

### 3.2 誕生日カレンダー
- 月別の誕生日一覧
- 来週・来月の誕生日表示
- 「今日が誕生日の人」のハイライト

### 3.3 リマインダー通知
- 誕生日の7日前・3日前・当日の通知
- プレゼント購入リマインダー

### 3.4 お祝い記録
- 毎年のお祝い方法・プレゼントの記録
- 相手の反応・感想メモ
- 過去のお祝い履歴の参照

### 3.5 グループお祝い計画（Phase 3）
- グループでのサプライズ計画
- 費用分担の管理

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 通知 | ブラウザ通知によるリマインダー |
| データ保存 | ユーザーごとのデータ永続化 |
| プライバシー | 連絡先情報の適切な保護 |
| パフォーマンス | カレンダー表示は2秒以内 |
