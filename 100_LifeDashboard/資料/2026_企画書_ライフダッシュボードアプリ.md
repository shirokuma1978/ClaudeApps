# ライフダッシュボードアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

健康・仕事・趣味・お金など人生のあらゆる側面を一つの画面で俯瞰できれば、より良い意思決定ができる。ライフダッシュボードアプリは以下の課題を解決する。

- 人生の複数の領域（健康・仕事・お金・趣味等）を一元管理する
- 日々の記録をダッシュボードで可視化して人生全体を俯瞰する
- バランスの取れた生き方を実現するための指標を提供する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（自己管理・生産性向上を目指す人） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | ライフコーチとのデータ共有 |

---

## 3. 主要機能

### 3.1 ライフダッシュボード（MVP）
- 人生の8領域スコアの可視化
  - 健康・体
  - 仕事・キャリア
  - お金・資産
  - 人間関係・家族
  - 趣味・娯楽
  - 学習・成長
  - 心・精神
  - 環境・生活
- 各領域の現在スコア（1〜10）
- レーダーチャートでの全体像表示

### 3.2 日次記録
- 毎日の各領域スコアの入力
- 今日の良かったこと・改善点
- 気分・エネルギーレベルの記録
- 写真・メモの添付

### 3.3 週次・月次レビュー
- 週間・月間のスコア推移グラフ
- 各領域の傾向分析
- バランスの崩れた領域の警告
- 改善提案メモ

### 3.4 目標との連携（Phase 2）
- 各領域の目標設定
- 目標達成率のダッシュボード表示
- 他アプリとのデータ連携（習慣・健康・家計等）

### 3.5 コーチング連携（Phase 3）
- ライフコーチへのダッシュボード共有
- フィードバックの受け取り
- コーチとの月次レビュー管理

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| グラフ | レーダーチャート・時系列グラフ |
| データ保存 | ユーザーごとのデータ永続化 |
| エクスポート | 月次レポートのPDF出力 |
| パフォーマンス | ダッシュボード表示は3秒以内 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（ライフダッシュボード・今日のスコア）
├── 日次記録入力画面
├── 領域別詳細・履歴画面
├── 週次レビュー画面
├── 月次レポート画面
├── 目標管理画面
└── 設定画面
```
