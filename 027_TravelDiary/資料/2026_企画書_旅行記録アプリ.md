# 旅行記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

旅行の思い出は写真だけでは伝わらない感動・体験がある。旅行記録アプリは以下の課題を解決する。

- 旅行中・旅行後の記録を日記形式で残す
- 訪問した場所・スポットをマップで可視化する
- 旅行の費用・日程を管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 同行者との旅行記録共有 |

---

## 3. 主要機能

### 3.1 旅行記録機能（MVP）
- 旅行の登録・編集・削除
- 登録項目：
  - 旅行タイトル
  - 目的地（国・都市・地域）
  - 旅行期間（開始日・終了日）
  - 旅行の種類（国内・海外・一人旅・家族旅行等）
  - 総費用
  - 評価・満足度
  - 旅行日記（日別のできごと）
  - 写真・動画

### 3.2 スポット記録
- 訪問したスポットの記録
- スポットの評価・コメント
- カテゴリ（観光地・グルメ・ホテルなど）

### 3.3 旅行日記機能
- 日別の旅行日記
- テキスト・写真の混合投稿
- タイムライン形式の表示

### 3.4 統計・可視化
- 訪問した都道府県・国のマップ
- 旅行回数・日数の統計
- 費用の傾向分析

### 3.5 旅行計画機能（Phase 3）
- 旅行前の計画作成
- 持ち物チェックリスト
- ルート計画

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 写真 | 複数写真のアップロード・ギャラリー表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| オフライン | 旅行中の入力をオフラインでも可能（将来） |
| 地図 | SVGマップによる訪問地の塗りつぶし |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近の旅行・次の旅行予定）

├── 旅行詳細・日記画面
├── 訪問マップ画面（都道府県・世界地図）
├── 統計画面

```
