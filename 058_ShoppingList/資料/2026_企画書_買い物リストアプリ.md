# 買い物リストアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

買い物リストはシンプルなニーズだが、家族間での共有・店舗別整理・常備品管理が課題である。買い物リストアプリは以下の課題を解決する。

- 家族間でリアルタイムに共有できる買い物リストを提供する
- 店舗別・カテゴリ別に整理してスムーズな買い物をサポートする
- 常備品を登録して定期的な補充リストを自動生成する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族全員でのリアルタイム共有 |

---

## 3. 主要機能

### 3.1 買い物リスト機能（MVP）
- アイテムの追加・編集・削除
- 登録項目：
  - 商品名
  - カテゴリ（野菜・肉・日用品等）
  - 数量・単位
  - 購入店舗
  - 優先度
  - メモ（ブランド指定等）

### 3.2 チェック機能
- 買ったものをチェックオフ
- チェック済みアイテムの非表示/表示切り替え
- カート投入順の並び替え

### 3.3 カテゴリ・通路別整理
- スーパーの売り場順に並び替え
- 店舗別のリスト管理

### 3.4 常備品管理
- 常備品リストの作成
- 在庫切れ時の自動追加設定

### 3.5 家族共有（Phase 3）
- リアルタイムの更新共有
- 買い物担当者の設定
- 完了通知

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| オフライン | 店舗内でのオフライン表示対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| リアルタイム | 家族間のリアルタイム共有（Phase 3） |
| パフォーマンス | チェック操作は即時 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（現在のリスト・完了済み）
├── アイテム追加・編集画面
├── カテゴリ別一覧
├── 常備品管理画面
└── 設定画面（店舗設定・共有設定）
```
