# 割り勘計算アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

グループでの旅行・食事・イベントで費用を精算する際、複雑な割り勘計算が面倒になる。割り勘計算アプリは以下の課題を解決する。

- 複数人・複数支払いの清算を自動計算して最小送金回数で精算する
- 旅行・食事会などのイベントごとに費用を管理する
- 参加者との清算状況を記録して支払い漏れを防ぐ

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用・グループイベントの幹事） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | グループメンバー全員での共同管理 |

---

## 3. 主要機能

### 3.1 イベント管理機能（MVP）
- イベントの作成・管理
- 参加者リストの管理
- 費用の追加・編集

### 3.2 費用入力・割り勘計算
- 費用の登録（誰が何を支払ったか）
- 費用の按分設定（均等・個別・パーセンテージ）
- 自動清算計算（最小送金回数）

### 3.3 清算状況管理
- 誰が誰にいくら払うかの一覧表示
- 支払い完了チェック
- 残債の追跡

### 3.4 履歴管理
- 過去のイベント履歴
- 参加者別の支払い実績
- よく一緒にいく人の管理

### 3.5 グループ共有（Phase 3）
- イベントへの複数ユーザーの招待
- リアルタイムでの費用共同入力
- QRコード・URLでの参加招待

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 計算精度 | 円単位での精確な計算 |
| データ保存 | ユーザーごとのデータ永続化 |
| 共有 | URLによるイベントシェア（Phase 3） |
| パフォーマンス | 計算は即時 |

---
