# ゴミ出しリマインダーアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

引っ越し先のゴミ回収日を把握・記憶するのは意外と難しく、出し忘れが発生しやすい。ゴミ出しリマインダーアプリは以下の課題を解決する。

- 住んでいる地区のゴミ回収スケジュールを登録して毎回のリマインダーを設定する
- 「今日何ゴミの日？」を素早く確認できる
- 引っ越し・地区変更時のスケジュール更新を簡単に行える

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・シェアハウスでの共有 |

---

## 3. 主要機能

### 3.1 ゴミ出しスケジュール登録（MVP）
- ゴミ種別の登録
- 回収頻度の設定：
  - 毎週特定曜日
  - 隔週特定曜日
  - 月1回・月2回（第何週曜日等）
- 複数の回収ルールの管理

### 3.2 今日・明日のゴミ確認
- ホーム画面でのゴミ種別一目表示
- 明日出すゴミの前夜確認

### 3.3 リマインダー通知
- 前夜の通知（例：夜9時「明日は可燃ゴミです」）
- 当日朝の通知（例：朝7時）
- カスタム通知時刻の設定

### 3.4 カレンダービュー
- 月間カレンダーでのゴミ出し日表示
- 種別ごとの色分け表示
- 祝日・年末年始の回収変更メモ

### 3.5 地区ルール管理（Phase 3）
- 地区コードから回収スケジュールの自動取得
- 近隣住民との情報共有

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |
| 通知 | ブラウザ通知・PWA通知 |
| データ保存 | ユーザーごとのデータ永続化 |
| UI | シンプルで直感的なゴミ種別表示 |
| オフライン | スケジュール確認はオフラインでも可能 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今日・明日のゴミ種別）
├── スケジュール登録・編集画面
├── カレンダービュー
├── 通知設定画面
└── 設定画面（地区設定・時刻設定）
```
