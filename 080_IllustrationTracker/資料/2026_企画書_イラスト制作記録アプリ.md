# イラスト制作記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

イラストレーター・絵描きが制作した作品・制作時間・スキルの成長を記録・管理するためのアプリが求められている。イラスト制作記録アプリは以下の課題を解決する。

- 制作したイラストを記録してポートフォリオとして管理する
- 制作時間・使用ツールを記録して生産性を分析する
- 上達の軌跡を可視化してモチベーションを維持する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（アマチュア・プロイラストレーター） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 絵描き仲間との作品共有・フィードバック |

---

## 3. 主要機能

### 3.1 作品記録機能（MVP）
- イラストの登録・編集・削除
- 登録項目：
  - 作品タイトル
  - ジャンル（キャラクター・風景・概念等）
  - 制作日・公開日
  - 使用ツール（Procreate・CLIP STUDIO等）
  - 制作時間
  - キャンバスサイズ
  - テーマ・コンセプト
  - 作品画像
  - 感想・改善点

### 3.2 スキルトラッキング
- 描く練習の記録（人体・背景・色彩等）
- 上達したと感じるポイントのメモ
- 練習量と上達度の相関分析

### 3.3 制作時間管理
- 作品ごとの制作時間記録
- 月別の制作時間グラフ
- 制作速度の向上推移

### 3.4 ポートフォリオ管理（Phase 2）
- 公開用ポートフォリオの選定
- カテゴリ別の作品整理
- 代表作の設定

### 3.5 コミュニティ（Phase 3）
- 絵描き仲間との作品共有
- 相互フィードバック機能

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| 画像 | 高解像度イラストのアップロード・表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| ギャラリー | 美しいグリッドギャラリー表示 |
| パフォーマンス | 画像一覧は3秒以内 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近の作品・今月の制作数）
├── 作品ギャラリー画面
├── 作品登録・編集画面
├── スキルトラッキング画面
├── 制作時間グラフ画面
└── 設定画面
```
