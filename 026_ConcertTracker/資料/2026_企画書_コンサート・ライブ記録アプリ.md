# コンサート・ライブ記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

コンサートやライブは一期一会の体験であり、その記憶を大切に残したいというニーズがある。コンサート・ライブ記録アプリは以下の課題を解決する。

- 参加したライブ・コンサートの記録を蓄積し、一生の思い出を整理する
- 今後行きたいアーティスト・イベントのウィッシュリストを管理する
- 参加した会場・アーティストの統計を楽しむ

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | ライブ仲間との記録共有・一緒に行く計画 |

---

## 3. 主要機能

### 3.1 ライブ記録機能（MVP）
- ライブ参加記録の登録・編集・削除
- 登録項目：
  - アーティスト名
  - イベント名・ツアー名
  - 日付・時間
  - 会場名・場所
  - 座席（スタンド・アリーナ・スタジオ等）
  - チケット代
  - 評価（5段階）
  - セットリスト（任意）
  - 感想・思い出メモ
  - 写真

### 3.2 ウィッシュリスト
- 行きたいアーティスト・イベントの登録
- チケット発売情報のメモ
- 優先度設定

### 3.3 会場マップ・統計
- 行ったことのある会場の一覧
- 同じアーティストを何回見たか
- 年別参加本数の推移

### 3.4 セットリスト管理（Phase 2）
- 演奏曲のリスト記録
- 曲の感情メモ（特に印象的だった曲など）

### 3.5 ライブ仲間機能（Phase 3）
- 一緒に行った人の記録
- 友人とのセットリスト比較

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 画像 | 公演写真のアップロード・ギャラリー表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録・閲覧は3秒以内 |
| 地図 | 会場の地図表示（将来） |
