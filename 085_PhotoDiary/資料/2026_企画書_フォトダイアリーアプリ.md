# フォトダイアリーアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

毎日1枚の写真と一言で記録する「フォトダイアリー」は、継続しやすく日常の豊かさを発見できる習慣である。フォトダイアリーアプリは以下の課題を解決する。

- 毎日1枚の写真と一言で日常を記録する習慣を支援する
- 写真と日記を組み合わせた視覚的な記録を残す
- 1年後に見返せる素晴らしいタイムラインを作る

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・友人との日常共有 |

---

## 3. 主要機能

### 3.1 日次記録機能（MVP）
- 毎日の写真＋一言の投稿
- 記録項目：
  - 日付
  - 写真（1枚〜複数枚）
  - キャプション・一言日記
  - 気分スコア
  - 場所（任意）
  - タグ

### 3.2 フォトタイムライン
- 日付順の写真タイムライン
- 月別・年別のグリッド表示
- カレンダービュー（記録済み日が一目でわかる）

### 3.3 振り返り機能
- 「1年前の今日」の写真を表示
- 月別・季節別のスライドショー
- 記録連続日数（ストリーク）

### 3.4 写真アルバム機能（Phase 2）
- テーマ別アルバムの作成
- ベスト写真のコレクション

### 3.5 共有機能（Phase 3）
- 家族・友人との日常共有
- グループフォトダイアリー

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ優先、PCブラウザ対応 |
| 画像 | 高品質写真のアップロード |
| データ保存 | ユーザーごとのデータ永続化 |
| 通知 | 毎日の記録リマインダー |
| パフォーマンス | 写真グリッド表示は3秒以内 |
