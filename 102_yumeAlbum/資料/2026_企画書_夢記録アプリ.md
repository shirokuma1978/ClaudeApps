# 夢記録アプリ（yumeAlbum）企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

自然と忘れてしまう夢を、項目別にタップで記録していく実験的アプリ。
「何が溜まるかわからないけど記録してみる」感覚で続けることで、自分の夢の傾向や無意識のパターンが見えてくることを期待する。

- 文字入力を最小限にして記録のハードルを下げる
- タグ・ボタン選択でざっくり整理するだけでOK
- 積み重ねてアルバムのように見返すのが楽しい

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |

| Phase 3（拡張） | 夢の傾向分析・統計表示 |

## 3. 主要機能

### 3.1 記録機能（MVP）
- 思い出したタイミングでいつでも記録できる（朝に限定しない）
- 日付は自動付与
- 最初に「覚えてる / 覚えてない」を選択
- 「覚えてない」もデータとして記録可能（任意）
- 「覚えてる」を選んだ場合は以下のタグを選択

**記録項目（タグ選択式）**

| 項目 | 選択肢 |
|-----|------|
| 場所・舞台 | 知ってる場所 / 知らない場所 / 家 / 屋外 / 非現実的な場所 |
| 登場人物 | 家族 / 友人 / 知らない人 / 自分だけ / 有名人 |
| 感情 | 楽しい / 怖い / 悲しい / 不思議 / ほっこり / ドキドキ |
| ストーリー | あった / なかった / ぼんやり |
| 鮮明度 | はっきり覚えてる / ぼんやり / ほとんど忘れた |

- メモ欄（任意）：一言だけ内容を残したいときに自由入力

### 3.2 アルバム表示（ホーム）
- 記録一覧をカード形式で表示
- 感情タグで色分け
- 日付・タグでフィルタリング

### 3.3 分析機能（Phase 3）
- 夢を覚えている頻度（月別）
- よく出る感情タグのランキング
- よく出る登場人物・場所の傾向
- 鮮明度の推移

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |

| 記録タイミング | 思い出したときにいつでも（時間制限なし） |
| データ保存 | ユーザーごとのデータ永続化 |

## 5. 画面構成（概要）

```
├── 記録画面
│   ├── 覚えてる / 覚えてない 選択
│   ├── タグ選択（場所・登場人物・感情・ストーリー・鮮明度）
│   └── メモ欄（任意・一言自由入力）
├── アルバム画面（ホーム）
│   ├── 記録一覧（カード形式）
│   └── フィルター（日付・タグ）
├── 分析画面（Phase 3）
│   ├── 夢を覚えてる頻度グラフ
│   ├── 感情タグランキング
│   └── 登場人物・場所の傾向
└── 設定画面（Phase 2）
    └── プロフィール・ログイン
```
