# 作業時間記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

フリーランス・副業・在宅ワークが普及する中、作業時間の正確な把握は報酬計算・生産性分析に不可欠である。作業時間記録アプリは以下の課題を解決する。

- プロジェクト・クライアント別の作業時間を正確に記録する
- 時間単価から作業収益を自動計算する
- 自分の時間の使い方を可視化して生産性を改善する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |

| Phase 3（拡張） | チームでの工数管理 |

---

## 3. 主要機能

### 3.1 時間記録機能（MVP）
- 作業の開始・終了の記録（タイマー方式）
- 手動入力による記録追加
- 登録項目：
  - 日付・時間
  - プロジェクト・クライアント名
  - 作業内容
  - 時間単価（任意）
  - メモ

### 3.2 タイマー機能
- ワンクリックで開始/停止
- 現在の作業時間のリアルタイム表示
- 同時に複数タスクのトラッキング（将来）

### 3.3 プロジェクト・クライアント管理
- プロジェクトの作成・編集
- クライアント情報の登録
- プロジェクト別の累計時間・収益

### 3.4 レポート・請求書連携（Phase 2）
- 週次・月次の作業時間レポート
- 作業時間×単価の収益計算
- 請求書作成アプリへのデータ連携

### 3.5 生産性分析（Phase 3）
- 時間帯別の生産性分析
- タスク種別（開発・会議・雑務）の時間配分
- 目標時間との比較

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| タイマー精度 | 1分以内の精度 |
| データ保存 | ユーザーごとのデータ永続化 |
| エクスポート | CSV形式での書き出し |
| パフォーマンス | タイマー開始/停止は即時 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（本日の作業時間・タイマー）
├── 作業記録一覧（日付別）
├── プロジェクト管理画面
├── 週次・月次レポート画面
├── 収益計算画面
└── 設定画面（単価設定・カテゴリ管理）
```
