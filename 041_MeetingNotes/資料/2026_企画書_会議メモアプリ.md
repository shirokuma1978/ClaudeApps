# 会議メモアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

会議での議事録作成は重要だが、メモをとりながら議論に集中するのは難しい。会議メモアプリは以下の課題を解決する。

- 会議の議事録を効率的に作成・管理する
- アクションアイテムを自動で抽出・管理する
- 過去の会議記録を素早く検索・参照できる

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | チームでの議事録共有・共同編集 |

---

## 3. 主要機能

### 3.1 会議メモ作成機能（MVP）
- 会議の登録・編集・削除
- 登録項目：
  - 会議名・タイトル
  - 日時
  - 参加者
  - アジェンダ
  - 議事内容（マークダウン対応）
  - 決定事項
  - アクションアイテム（担当者・期限付き）
  - 次回の予定

### 3.2 テンプレート機能
- 会議種別のテンプレート（週次定例・1on1・キックオフ等）
- カスタムテンプレートの作成・保存

### 3.3 アクションアイテム管理
- 会議横断でのアクションアイテム一覧
- 担当者・期限の管理
- 完了チェック機能

### 3.4 検索・参照機能
- キーワード全文検索
- 日付・参加者でのフィルタリング
- プロジェクト別の会議グルーピング

### 3.5 チーム共有（Phase 3）
- 議事録のURL共有
- リアルタイム共同編集

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| エディタ | マークダウン対応エディタ |
| データ保存 | ユーザーごとのデータ永続化 |
| 検索速度 | 全文検索は3秒以内 |
| エクスポート | PDF・テキスト形式でのエクスポート |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近の会議・未完了アクションアイテム）
├── 会議メモ作成画面（マークダウンエディタ）
├── 会議一覧・検索画面
├── 会議詳細画面
├── アクションアイテム管理画面
└── 設定画面（テンプレート管理）
```
