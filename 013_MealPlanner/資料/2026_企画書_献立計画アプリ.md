# 献立計画アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

毎日の「今日の夕ご飯何にしよう」という悩みは多くの家庭で発生する。献立計画アプリは以下の課題を解決する。

- 1週間の献立をまとめて計画し、「今日の夕食」に悩む時間を削減する
- 栄養バランスを考慮した献立作成を支援する
- 献立に合わせた買い物リストを自動生成し、買い物を効率化する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族での献立共有・分担計画 |

---

## 3. 主要機能

### 3.1 献立カレンダー機能（MVP）
- 週間・月間カレンダー上での献立計画
- 朝食・昼食・夕食の設定
- レシピからの献立設定
- フリー入力による献立メモ

### 3.2 レシピ連携機能
- 登録済みレシピから選択して献立に設定
- レシピ候補のランダム提案
- 最近使っていないレシピの優先提案

### 3.3 買い物リスト自動生成
- 週間献立から必要な食材を自動集計
- 冷蔵庫在庫との突合で不足分のみ表示
- カテゴリ別・店別の整理

### 3.4 栄養バランス可視化（Phase 2）
- 主食・主菜・副菜のバランスチェック
- 野菜・たんぱく質の摂取バランス
- 偏りのある献立への警告

### 3.5 献立履歴・ローテーション（Phase 3）
- 過去の献立履歴の参照
- 同じ料理が続かないようなアドバイス

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | 週間カレンダーはPC表示を優先 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 献立設定・更新は2秒以内 |
| 連携 | レシピ管理・食材在庫管理アプリと連携 |
