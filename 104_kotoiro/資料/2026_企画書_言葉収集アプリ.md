# 言葉収集アプリ（kotoiro）企画書

**バージョン:** 1.1
**作成日:** 2026年3月27日
**更新日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

日常の中で出会った面白い言葉、胸に刺さるフレーズ、美しい日本語表現を記録していく、自分だけの言葉の引き出しアプリ。

- 「あの言葉なんだっけ」をなくす。物忘れする前にさっと記録
- 溜めた言葉をランダムに眺めて、ぼーっと楽しめる
- 必要なときにさっと取り出せる自分専用の言葉コレクション

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録によるマルチデバイス対応 |
| Phase 3（拡張） | 言葉の共有・他のユーザーのコレクションを見る |

---

## 3. 主要機能

### 3.1 記録機能（MVP）
- 言葉・フレーズを素早く登録
- 登録項目：
  - 言葉・フレーズ（本文）
  - 出典（本 / 会話 / テレビ / SNS / その他）
  - 日付（自動付与）
  - メモ・背景（任意）：意味や背景を書いておくとキャラモードの精度が上がる
  - ランダム表示するかどうかのチェック

### 3.2 ながめるモード（MVP）
- 登録済みの言葉をランダムにゆっくり表示
- ランダム表示チェックが入っているものだけ流れる
- スクリーンセーバーのようにぼーっと眺められるUI
- 意味はないけどずっと見ていられる感覚

### 3.3 キャラモード（Phase 3）
- 歴史上の人物・文豪・アニメキャラなどを選択
- 自分のコレクションの言葉を選んだキャラのフィルターを通してAIが解釈・変換
- 「この言葉をチャーリーブラウンが言ったら？」「安西先生が言ったら？」
- キャラになりきった言葉をながめるモードで流せる
- 対象キャラ例：夏目漱石・太宰治・松尾芭蕉・チャーリーブラウン・アニメキャラ など

### 3.4 検索・探す機能
- キーワード検索
- 出典別フィルター（本・会話・テレビ・SNS）
- 登録日順・ランダム順で表示

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |
| 操作性 | 気になった瞬間にさっと記録できる手軽さ |
| ながめるモード | 言葉がふわっと切り替わるなめらかなアニメーション |
| データ保存 | ユーザーごとのデータ永続化 |

## 5. 画面構成（概要）

```
├── ながめるモード（ホーム）
│   └── 言葉がランダムにふわっと流れる
├── 記録画面
│   ├── 言葉・フレーズ入力
│   ├── 出典選択
│   ├── メモ（任意）
│   └── ランダム表示チェック
├── 一覧・検索画面
│   ├── キーワード検索
│   └── 出典フィルター
└── 設定画面（Phase 2）
    └── プロフィール・ログイン
```
