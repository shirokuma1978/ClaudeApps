# タスク管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

日常のタスクや仕事の作業項目を効率的に管理することで、生産性を高め、やり忘れを防ぐ。タスク管理アプリは以下の課題を解決する。

- タスクを一元管理し、何をすべきかを常に明確にする
- 優先度・期限を設定して重要なタスクを見逃さない
- 完了記録を積み重ねることで達成感と生産性の可視化を実現する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | チームでのタスク共有・割り当て |

---

## 3. 主要機能

### 3.1 タスク登録機能（MVP）
- タスクの作成・編集・削除
- 登録項目：
  - タスク名
  - 説明
  - 期限日時
  - 優先度（高・中・低）
  - カテゴリ・プロジェクト
  - サブタスク

### 3.2 ビュー切り替え機能
- リストビュー
- カンバンボードビュー（ToDo・進行中・完了）
- カレンダービュー（期限ベース）

### 3.3 フィルタリング・検索機能
- 期限・優先度・カテゴリでのフィルター
- キーワード検索
- 完了済みタスクの表示・非表示

### 3.4 繰り返しタスク機能
- 定期タスクの設定（毎日・毎週・毎月）
- 自動タスク生成

### 3.5 統計・レポート機能（Phase 3）
- 完了タスク数の推移
- カテゴリ別の作業時間分析
- 生産性スコアの計算

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | デスクトップでのカンバンビューを優先 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | タスク操作は1秒以内 |
| ドラッグ&ドロップ | カンバンでのタスク移動 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ダッシュボード（今日のタスク・直近の期限）
├── タスク一覧（リスト/カンバン/カレンダー切り替え）
├── タスク作成・編集画面
├── タスク詳細画面
├── プロジェクト管理画面
├── 検索・フィルター画面
└── 設定画面（カテゴリ管理・通知設定）
```
