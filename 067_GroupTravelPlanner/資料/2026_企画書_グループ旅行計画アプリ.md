# グループ旅行計画アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

グループ旅行は計画・費用分担・スケジュール調整が複雑で、LINEやスプレッドシートで管理するのが煩雑である。グループ旅行計画アプリは以下の課題を解決する。

- 旅行の計画・スケジュール・費用をグループ全員で共有・管理する
- 旅行費用の精算を自動計算してトラブルを防ぐ
- 旅行の思い出・写真を全員で共有できる

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 旅行計画の幹事（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | グループ全員での参加・共同編集 |

---

## 3. 主要機能

### 3.1 旅行プロジェクト管理（MVP）
- 旅行の基本情報登録
- 参加者リストの管理
- 行程表の作成（日程・訪問地・宿泊地）

### 3.2 費用管理・精算
- 立替費用の記録（誰が何をいくら払ったか）
- 費用の自動割り勘計算
- 最終精算額の表示

### 3.3 スケジュール管理
- 出発・観光・食事・宿泊のタイムライン
- 交通手段・時刻の記録
- チェックリスト（持ち物・予約確認）

### 3.4 投票機能
- 候補地・候補ホテルの投票
- 出発日時の調整（Doodle的機能）

### 3.5 思い出共有（Phase 3）
- 参加者全員の写真をアップロード・共有
- 旅行アルバムの自動生成

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 共有 | URLによるグループ参加・招待 |
| データ保存 | ユーザーごとのデータ永続化 |
| リアルタイム | グループ内の変更をリアルタイム共有 |
| 精算計算 | 最小送金回数での精算計算 |
