# 家族の記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

子育て中の親にとって、子供の成長記録・行事・健康管理を一元管理したいというニーズは大きい。家族の記録アプリは以下の課題を解決する。

- 子供の成長記録（身長・体重・できた事）を記録・可視化する
- 家族の行事・記念日を管理して大切な日を逃さない
- 家族の健康・医療情報を管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人・保護者利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族全員でのデータ共有 |

---

## 3. 主要機能

### 3.1 家族メンバー管理（MVP）
- 家族構成の登録
- メンバー別のプロフィール管理

### 3.2 成長記録機能（子供向け）
- 身長・体重の定期記録
- 成長グラフの表示
- はじめての体験（初めて立った・歩いた等）の記録
- 写真アルバム

### 3.3 行事・記念日管理
- 家族の記念日カレンダー
- 七五三・入学・卒業などの行事記録
- 予定の家族間共有

### 3.4 家族健康管理
- 各メンバーの健康診断記録
- ワクチン接種記録
- かかりつけ医・病院情報

### 3.5 家族ニュースレター（Phase 3）
- 月次の家族ニュースレター自動生成
- 祖父母等への共有機能

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 画像 | 子供の写真・大量アップロード対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| プライバシー | 子供の情報の厳格な保護 |
| 通知 | 行事・記念日のリマインダー |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（家族の最近のできごと・今週の予定）
├── メンバー別プロフィール画面
├── 成長記録・グラフ画面
├── 写真アルバム画面
├── 行事・記念日カレンダー
├── 健康管理画面
└── 設定画面
```
