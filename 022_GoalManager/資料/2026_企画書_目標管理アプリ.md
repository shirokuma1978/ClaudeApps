# 目標管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

明確な目標設定と進捗管理は、自己実現の基盤となる。目標管理アプリは以下の課題を解決する。

- 年間・月間・週間の目標をツリー構造で管理し、大きな目標を行動レベルに分解する
- 目標の進捗を定期的に振り返り、修正サイクルを回す
- 達成した目標を記録して自己効力感を高める

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | コーチ・メンターとの目標共有 |

---

## 3. 主要機能

### 3.1 目標登録機能（MVP）
- 目標の作成・編集・削除
- 登録項目：
  - 目標名
  - 期間（長期・中期・短期）
  - カテゴリ（仕事・健康・学習・趣味など）
  - 達成指標（KPI）
  - 期限
  - 現在の進捗（%）
  - メモ・理由（WHY）

### 3.2 OKR・目標ブレイクダウン
- 大目標（Objective）を小目標（Key Result）に分解
- 階層構造での目標管理
- 親子目標のリンク

### 3.3 振り返り機能
- 週次・月次の振り返りテンプレート
- 目標に対する達成・未達の記録
- 次のアクションの設定

### 3.4 進捗可視化
- 目標ごとの進捗バー
- ロードマップ形式の表示
- 達成済み目標のタイムライン

### 3.5 目標レポート（Phase 3）
- 年間目標達成レポートの生成
- コーチへの共有機能

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| UI | 階層構造の目標を視覚的に把握しやすい設計 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 目標ツリーの表示は3秒以内 |
| リマインダー | 振り返りタイミングの通知 |

├── 振り返り入力画面
├── 統計・達成履歴画面
