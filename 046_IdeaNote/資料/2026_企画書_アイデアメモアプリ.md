# アイデアメモアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

閃いたアイデアは素早く記録しないと消えてしまう。アイデアメモアプリは以下の課題を解決する。

- 思いついたアイデアをすぐに記録できる（スピード重視）
- アイデアを整理・分類・発展させてプロジェクトに昇格させる
- 類似アイデアの検索・組み合わせで創造性を高める

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | チームでのアイデア共有・ブレスト |

---

## 3. 主要機能

### 3.1 アイデア記録機能（MVP）
- ワンタップでアイデア入力開始
- 登録項目：
  - タイトル
  - 内容（自由テキスト・マークダウン）
  - カテゴリ・タグ
  - ステータス（アイデア・検討中・実行中・完了・没）
  - 重要度

### 3.2 クイックキャプチャ
- ホーム画面から2タップで入力開始
- 音声入力での記録
- URLからの情報クリッピング

### 3.3 アイデア整理・発展
- マインドマップ形式での展開（Phase 2）
- 関連アイデアのリンク
- ブレインストーミングセッションの記録

### 3.4 検索・発見機能
- 全文検索・タグ検索
- ランダムアイデア表示（過去のアイデアとの出会い）
- 「今日の振り返り」機能

### 3.5 プロジェクト昇格（Phase 3）
- アイデアからプロジェクト・タスクへの変換
- プロジェクト管理アプリとの連携

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 入力速度 | アプリ起動からメモ入力まで3秒以内 |
| データ保存 | ユーザーごとのデータ永続化 |
| 全文検索 | 高速な全文検索 |
| UI | シンプルでクリエイティビティを刺激するデザイン |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近のアイデア・クイックキャプチャ）
├── アイデア一覧・検索画面
├── アイデア作成・編集画面
├── カテゴリ・タグ管理画面
├── 統計・振り返り画面
└── 設定画面
```
