# ランチ記録アプリ 企画書

**バージョン:** 2.0
**作成日:** 2026年3月25日
**更新日:** 2026年3月26日
**ステータス:** Phase 2 完成

---

## 1. 背景・目的

日々のランチをただ食べるだけでなく、記録・振り返ることで以下を実現する。

- お気に入りの店・メニューを蓄積し、迷ったときに参考にできる
- 食費の把握と管理
- 将来的に友人・同僚との体験共有ツールへ拡張

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 友人・同僚との共有 |

---

## 3. 主要機能

### 3.1 記録機能（MVP）
- 食事記録の登録・編集・削除
- 記録項目：
  - 日付・時間帯
  - 店名・メニュー名
  - 金額
  - 場所・エリア
  - 一緒に行った人
  - 評価（5段階）
  - メモ・コメント

### 3.2 一覧・検索機能
- 記録の一覧表示（日付順）
- エリア・評価・金額でのフィルタリング
- キーワード検索（店名・メニュー名）

### 3.3 集計・可視化機能
- 月別食費の集計
- よく行く店・エリアのランキング
- 平均単価の表示

### 3.4 認証機能（Phase 2・完成）
- ユーザー登録（名前・メールアドレス・パスワード）
- メールアドレス＋パスワードによるログイン
- JWTトークンによるセッション管理（30日間有効）
- ユーザーごとのデータ分離（自分の記録のみ表示・操作）

### 3.5 共有機能（Phase 3）
- 友人へのおすすめ共有
- 共有リスト作成

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先、PC対応も視野 |
| レスポンシブ | スマホ画面に最適化したUI |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録・一覧表示は3秒以内 |

```
├── ログイン画面           ← Phase 2 追加
├── ユーザー登録画面       ← Phase 2 追加
├── ホーム（最近の記録一覧）
├── 記録登録画面
├── 記録詳細画面
├── 検索・フィルター画面
├── 集計・統計画面
└── 設定画面（Phase 3: プロフィール・共有）
```
