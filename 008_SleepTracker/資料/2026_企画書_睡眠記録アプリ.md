# 睡眠記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

睡眠は健康の基盤であるが、自分の睡眠状態を客観的に把握している人は少ない。睡眠記録アプリは以下の課題を解決する。

- 就寝・起床時間、睡眠時間を記録し可視化する
- 睡眠の質（深い眠り・目覚めの爽快感）を評価し改善点を見つける
- 日中の活動・体調と睡眠の相関関係を分析する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族の睡眠管理（子供の就寝リズム確認等） |

---

## 3. 主要機能

### 3.1 睡眠記録機能（MVP）
- 睡眠の登録・編集・削除
- 登録項目：
  - 就寝時刻
  - 起床時刻
  - 睡眠の質（5段階）
  - 目覚めの状態（すっきり・眠い・疲れが残る）
  - メモ（夢・いびきなど）
  - 睡眠時間の自動計算

### 3.2 グラフ・可視化機能
- 週間・月間の睡眠時間グラフ
- 睡眠の質の推移
- 理想睡眠時間との比較

### 3.3 パターン分析機能
- 就寝・起床時刻の分布
- 曜日別の睡眠パターン
- 睡眠負債の計算

### 3.4 入眠サポート機能（Phase 2）
- 目標就寝時刻のリマインダー
- 快適な睡眠のためのヒント表示

### 3.5 相関分析機能（Phase 3）
- 運動・食事・ストレスとの相関
- 体調・気分との関連性分析

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |
| レスポンシブ | スマホでの素早い記録を優先 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録入力は1秒以内 |
| ウィジェット | ホーム画面からのクイック入力（将来） |

```
├── カレンダービュー
├── 週間・月間グラフ画面
├── 分析・パターン画面

```
