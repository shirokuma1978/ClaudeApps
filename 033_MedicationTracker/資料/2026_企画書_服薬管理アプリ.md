# 服薬管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

定期的な服薬は医療治療の基本であるが、飲み忘れや飲み過ぎは健康に影響する。服薬管理アプリは以下の課題を解決する。

- 服薬のリマインダーで飲み忘れを防ぐ
- 薬の在庫を管理して補充タイミングを把握する
- 服薬記録を蓄積して医療機関への報告に役立てる

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 介護者による家族の服薬管理 |

---

## 3. 主要機能

### 3.1 薬の登録機能（MVP）
- 薬の登録・編集・削除
- 登録項目：
  - 薬品名
  - 用量・単位（錠・カプセル・ml等）
  - 服薬タイミング（朝・昼・夕・就寝前）
  - 服薬頻度（毎日・週N回・頓服）
  - 処方医療機関
  - 処方日・処方量
  - 色・形状メモ
  - 注意事項

### 3.2 服薬チェック機能
- 本日の服薬リスト表示
- 服薬完了のチェック（ワンタップ）
- 飲み忘れアラートの表示

### 3.3 リマインダー機能（Phase 2）
- 設定時刻でのブラウザ通知
- 飲み忘れ時の再通知

### 3.4 在庫管理
- 残薬数の管理
- 補充（処方箋受け取り）のタイミング通知

### 3.5 服薬記録レポート（Phase 3）
- 服薬率のレポート生成
- 医師への報告用サマリー

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |
| セキュリティ | 医療情報の適切な保護 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | チェック操作は1秒以内 |
| 通知 | ブラウザ通知によるリマインダー |
