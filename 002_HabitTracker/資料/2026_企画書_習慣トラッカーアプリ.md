# 習慣トラッカーアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

良い習慣を身につけることは、人生の質を高める上で最も重要なことの一つだが、継続することが難しい。習慣トラッカーアプリは以下の課題を解決する。

- 毎日の習慣の実施状況を可視化し、継続のモチベーションを維持する
- 「連続記録（ストリーク）」を表示することで、途切れさせたくない心理を活用する
- 週・月単位での達成率を振り返り、改善サイクルを回す

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 友人・家族との習慣共有・励まし合い |

---

## 3. 主要機能

### 3.1 習慣登録機能（MVP）
- 習慣の登録・編集・削除
- 登録項目：
  - 習慣名
  - 頻度（毎日・週N回・特定曜日）
  - カテゴリ（健康・学習・仕事など）
  - 目標時間（任意）
  - アイコン・カラー設定

### 3.2 チェックイン機能
- 当日の習慣一覧表示
- 完了チェック（ワンタップで記録）
- 一部達成・スキップの記録

### 3.3 ストリーク・統計機能
- 連続達成日数の表示
- 週間・月間の達成カレンダー
- 達成率のグラフ表示
- 習慣ごとの統計

### 3.4 リマインダー機能（Phase 2）
- 習慣ごとのリマインダー設定
- ブラウザ通知

### 3.5 振り返りレポート（Phase 3）
- 週次・月次レポートの自動生成
- 習慣の相関分析

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先、PC対応も視野 |
| レスポンシブ | スマホ画面に最適化したUI |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | チェックイン操作は1秒以内 |
| オフライン対応 | Service Workerによるオフライン記録（将来） |

## 5. 画面構成（概要）

```

├── 統計・グラフ画面
├── 習慣一覧管理画面
└── 設定画面（リマインダー・プロフィール）
```
