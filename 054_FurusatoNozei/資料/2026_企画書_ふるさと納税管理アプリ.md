# ふるさと納税管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

ふるさと納税は節税効果の高い制度だが、上限額の把握・寄付記録・返礼品管理・確定申告が複雑である。ふるさと納税管理アプリは以下の課題を解決する。

- ふるさと納税の寄付記録・返礼品管理を一元化する
- 寄付上限額の目安計算と残枠の把握をする
- 確定申告・ワンストップ特例の提出状況を管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 夫婦での寄付・控除管理 |

---

## 3. 主要機能

### 3.1 寄付記録機能（MVP）
- 寄付の登録・編集・削除
- 登録項目：
  - 寄付先（自治体名）
  - 寄付金額
  - 寄付日
  - 返礼品名・カテゴリ
  - 返礼品の到着状況
  - サイト（楽天・ふるなびなど）
  - ワンストップ特例の申請状況

### 3.2 控除額シミュレーター
- 年収・家族構成から寄付上限額を計算
- 今年の寄付済み金額と残枠の表示

### 3.3 返礼品管理
- 返礼品の到着予定・到着確認
- カテゴリ別の返礼品一覧
- お気に入り自治体・返礼品の記録

### 3.4 確定申告管理（Phase 2）
- ワンストップ特例申請書の送付状況
- 確定申告用の寄付金控除証明書の管理
- 年度別の寄付記録の出力

### 3.5 来年計画（Phase 3）
- リピートしたい返礼品のウィッシュリスト
- 来年の寄付計画の立案

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| 計算精度 | 控除上限額の正確な計算 |
| パフォーマンス | 計算・表示は2秒以内 |
| 年度管理 | 複数年度の記録管理 |

```
├── 寄付一覧画面（年度別）

├── 返礼品管理画面
├── 確定申告管理画面

```
