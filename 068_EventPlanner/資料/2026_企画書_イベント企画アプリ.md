# イベント企画アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

飲み会・パーティー・同窓会などのイベント幹事は、準備・集金・出欠管理が多岐にわたり負担が大きい。イベント企画アプリは以下の課題を解決する。

- イベントの企画から当日・事後フォローまでを一元管理する
- 参加者の出欠確認・集金を効率化する
- 過去のイベント記録を活用して次回の準備を効率化する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | イベント幹事（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 参加者全員での共同管理 |

---

## 3. 主要機能

### 3.1 イベント管理機能（MVP）
- イベントの作成・編集・削除
- 登録項目：
  - イベント名・概要
  - 日時・場所
  - 参加者リスト
  - 参加費・集金方法
  - 準備タスクリスト
  - ドレスコード・持ち物

### 3.2 出欠管理
- 参加者への出欠確認（URL共有）
- 回答状況の一覧表示
- 参加/不参加/未回答のカウント

### 3.3 集金管理
- 参加費の記録・集金状況の管理
- 支払い済み/未払いの管理
- 残金・過不足の計算

### 3.4 準備タスク管理
- イベント準備チェックリスト
- 役割分担の設定
- 期限付きタスク管理

### 3.5 事後管理（Phase 3）
- イベント報告の作成
- 写真共有
- 次回開催の計画

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| URL共有 | 参加者への招待URLの発行 |
| データ保存 | ユーザーごとのデータ永続化 |
| 計算 | 集金の自動計算・過不足管理 |
| パフォーマンス | 一覧表示は3秒以内 |
