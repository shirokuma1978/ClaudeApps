# ペット管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

ペットの健康管理・病院記録・日常ケアを適切に管理することは、愛するペットの長寿・健康維持に重要である。ペット管理アプリは以下の課題を解決する。

- ペットの基本情報・健康記録・病院履歴を一元管理する
- ワクチン接種・健康診断の期限をアラートで通知する
- 日々のペットの様子を記録して成長・健康状態の変化を把握する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・多頭飼い・ペットシッターとの共有 |

---

## 3. 主要機能

### 3.1 ペット基本情報管理（MVP）
- ペットの登録・編集
- 登録項目：
  - 名前・種類・品種
  - 誕生日・年齢
  - 性別・体重
  - カラー・特徴
  - マイクロチップ番号
  - 写真

### 3.2 健康記録機能
- 体重・体温の定期記録
- 食事量・水分量の記録
- 日々の様子メモ・写真

### 3.3 医療・ワクチン管理
- ワクチン接種記録・次回接種日のアラート
- 病院受診記録（日付・症状・診断・処方）
- かかりつけ医の情報管理
- 薬の服薬管理

### 3.4 ケアスケジュール
- トリミング・爪切りの記録・次回リマインダー
- フィラリア・ノミダニ予防のスケジュール管理

### 3.5 成長記録・思い出（Phase 3）
- 写真アルバムでの成長記録
- 記念日の管理（お誕生日・家族になった日）

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| 画像 | ペット写真のアップロード・ギャラリー |
| データ保存 | ユーザーごとのデータ永続化 |
| 通知 | ワクチン・ケアのリマインダー |
| 複数ペット | 複数のペットを管理可能 |
