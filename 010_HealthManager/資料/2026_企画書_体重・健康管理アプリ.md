# 体重・健康管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

体重・BMI・体脂肪率などの身体データを継続的に記録・管理することで、健康目標の達成を支援する。体重・健康管理アプリは以下の課題を解決する。

- 毎日の体重を記録し、長期的なトレンドを可視化する
- BMI・体脂肪率などの健康指標を総合的に管理する
- 目標体重の設定と現在地の差を常に把握できる

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族の健康データ管理 |

---

## 3. 主要機能

### 3.1 体重記録機能（MVP）
- 体重・身体データの登録・編集・削除
- 登録項目：
  - 日付
  - 体重
  - 体脂肪率
  - 筋肉量
  - 基礎代謝
  - BMI（自動計算）
  - 体調メモ

### 3.2 グラフ・可視化機能
- 体重推移グラフ（週・月・年）
- BMI推移グラフ
- 目標体重との差の表示
- 移動平均線の表示

### 3.3 目標管理機能
- 目標体重・達成期限の設定
- 現在のペースでの達成予測日
- マイルストーン達成通知

### 3.4 健康指標管理
- 血圧・脈拍の記録
- 体調チェックリスト
- 生理周期との連動（女性ユーザー向け）

### 3.5 健康レポート（Phase 3）
- 医師への共有用レポート生成
- 健康診断結果の記録・比較

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |
| レスポンシブ | 毎朝の計量を素早く入力できるUI |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | データ入力は1秒以内 |
| プライバシー | 健康データの厳格な保護 |

├── 健康指標一覧画面
