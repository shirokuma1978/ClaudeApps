# 副業収入管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

副業・複業が一般化する中、複数の収入源を管理することが複雑になっている。副業収入管理アプリは以下の課題を解決する。

- 複数の副業収入源を一元管理して全体像を把握する
- 確定申告に必要な副業収入データを整理する
- 副業の時給換算・生産性を分析して最も効率の良い活動に集中する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（副業・複業している個人） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 副業仲間との情報共有・相談 |

---

## 3. 主要機能

### 3.1 収入記録機能（MVP）
- 収入の登録・編集・削除
- 登録項目：
  - 日付
  - 金額
  - 収入源（クライアント・プラットフォーム名）
  - 副業カテゴリ（ライター・エンジニア・デザイン等）
  - 作業時間
  - 時給換算（自動計算）
  - メモ

### 3.2 収入源管理
- 副業の種類・プラットフォームの管理
- 各収入源のパフォーマンス比較
- 収入源ごとのアクティブ/停止状態

### 3.3 月次・年次集計
- 月別・収入源別の収入グラフ
- 時給換算ランキング
- 年収シミュレーション

### 3.4 確定申告管理（Phase 2）
- 雑所得・事業所得の区分管理
- 経費との差し引き計算
- 確定申告用データの出力（CSV）

### 3.5 目標管理（Phase 3）
- 月次・年次の収入目標設定
- 目標達成率の可視化
- 副業別の成長分析

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| データ保存 | ユーザーごとのデータ永続化 |
| セキュリティ | 収入情報の適切な保護 |
| パフォーマンス | 集計・グラフ表示は3秒以内 |
| エクスポート | CSV形式での書き出し |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今月の収入サマリー・副業別内訳）
├── 収入記録入力画面
├── 収入一覧画面（月別）
├── 収入源管理画面
├── 統計・グラフ画面
├── 確定申告用データ出力画面
└── 設定画面
```
