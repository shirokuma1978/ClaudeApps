# 資格取得管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

資格取得は長期的な勉強計画が必要で、複数の資格を並行して管理するのが困難である。資格取得管理アプリは以下の課題を解決する。

- 取得したい資格・取得済み資格を一元管理する
- 試験日から逆算した学習計画を作成し、進捗を管理する
- 学習記録を蓄積してモチベーションを維持する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 同じ資格を目指す仲間との情報共有 |

---

## 3. 主要機能

### 3.1 資格管理機能（MVP）
- 資格の登録・編集・削除
- 登録項目：
  - 資格名
  - 分野（IT・語学・ビジネス・国家資格など）
  - ステータス（取得済み・勉強中・検討中）
  - 取得日（取得済みの場合）
  - 試験日・申込期限
  - 合格率・難易度
  - 参考書・教材情報
  - メモ

### 3.2 学習計画機能
- 試験日から逆算した学習スケジュール作成
- 週次学習目標の設定
- タスクベースの進捗管理

### 3.3 学習記録機能
- 日次学習時間の記録
- 学習内容（読んだ章・解いた問題数）の記録
- 正答率・習熟度の自己評価

### 3.4 試験結果管理
- 模擬試験・過去問の結果記録
- スコアの推移グラフ
- 弱点分野の分析

### 3.5 資格ポートフォリオ（Phase 3）
- 取得済み資格の一覧表示（証明書写真添付）
- キャリア目標との紐付け

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | 勉強中のスマホ入力を考慮 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録入力は2秒以内 |
| リマインダー | 試験日・申込期限のアラート |

```
├── 学習記録入力画面

├── 統計・グラフ画面

```
