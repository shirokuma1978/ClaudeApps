# 家事管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

家事の偏り・やり忘れは家庭内の問題になりやすい。家事管理アプリは以下の課題を解決する。

- 家事タスクをリスト化して見える化する
- 定期的な家事（週次・月次）のリマインダーを設定する
- 家族間での家事分担を公平に管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族全員での家事分担・確認 |

---

## 3. 主要機能

### 3.1 家事タスク管理（MVP）
- 家事タスクの登録・編集・削除
- 登録項目：
  - 家事名
  - カテゴリ（掃除・洗濯・料理・ゴミ出し等）
  - 頻度（毎日・週次・月次・年次）
  - 担当者（自分・パートナー・子供）
  - 所要時間
  - 最終実施日
  - 次回実施日（自動計算）

### 3.2 今日の家事リスト
- 今日やるべき家事の表示
- 完了チェック機能
- 未完了家事の翌日繰り越し

### 3.3 家事分担管理（Phase 2）
- メンバー別の担当家事一覧
- 家事負担度の可視化
- バランスの改善提案

### 3.4 家事ルーティン管理
- 朝・夜のルーティン設定
- 季節の大掃除リスト

### 3.5 家族共有（Phase 3）
- リアルタイムでの家事完了共有
- 家事完了のお礼・ポジティブフィードバック機能

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | スマホでの素早いチェックを優先 |
| データ保存 | ユーザーごとのデータ永続化 |
| 通知 | 定期家事のリマインダー |
| パフォーマンス | チェック操作は即時 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今日の家事・未完了タスク）

├── 家事登録・編集画面
├── 家事分担管理画面
├── カレンダービュー

```
