# デザインインスピレーション管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

デザイナー・クリエイターにとって、インスピレーション素材の収集・整理は創作活動の基盤となる。デザインインスピレーション管理アプリは以下の課題を解決する。

- 見つけたデザインサンプル・参考素材を一元管理する
- プロジェクト・テーマ別に整理して作業時にすぐ参照できる
- お気に入りの色・フォント・パターンをストックする

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（デザイナー・クリエイター） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | デザインチームでのインスピレーション共有 |

---

## 3. 主要機能

### 3.1 インスピレーション登録機能（MVP）
- インスピレーション素材の登録
- 登録項目：
  - 画像（URL・アップロード）
  - タイトル・説明
  - カテゴリ（UI/UX・タイポグラフィ・カラー等）
  - タグ
  - 参照URL・出典
  - 評価・ノート

### 3.2 カテゴリ・コレクション管理
- プロジェクト別のコレクション
- テーマ・スタイル別の整理
- フォルダ型管理

### 3.3 カラー・フォント管理
- お気に入りカラーパレットの保存
- フォントの記録（フォント名・使用例）
- デザインスタイル別の分類

### 3.4 検索・発見機能
- キーワード・タグ・カテゴリでの検索
- 類似デザインの表示
- ランダムインスピレーション表示

### 3.5 チーム共有（Phase 3）
- チームへのコレクション共有
- コメント・フィードバック機能

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| 画像 | 高品質画像の効率的な表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 画像グリッド表示は3秒以内 |
| URL取込 | URLからのOGP画像・情報自動取得 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（最近追加・コレクション一覧）
├── インスピレーション一覧（グリッド表示）
├── 登録・編集画面
├── コレクション管理画面
├── カラー・フォントライブラリ画面
├── 検索画面
└── 設定画面
```
