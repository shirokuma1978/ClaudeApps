# 公園宝探しアプリ（KouenQuest）企画書

**バージョン:** 1.8
**作成日:** 2026年3月27日
**更新日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

親が地図上に宝をセットして、親子で一緒にレーダーを見ながら公園の中を歩き回って宝を探す、リアル冒険アプリ。
ドラゴンボールのドラゴンレーダーをイメージしたUIで、公園が冒険の舞台になる。

* 親子で一緒に体を動かしながら遊べる
* 7個の宝を集めると神龍が現れ、子どものお願いごとが1つ叶う
* 子どもが「欲しいものリスト」を持ち、親が承認する仕組みで自然な親子の約束が生まれる
* 娘が成長したら自分でレーダーを持って走り回れるようになる

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
| --- | --- |
| Phase 1（初期） | 親子（2歳〜）で一緒に遊ぶ |
| Phase 2（拡張） | 子どもが自分でレーダーを操作して遊ぶ（4歳〜） |
| Phase 3（コミュニティ） | 他の親子が設置した宝を探しに行く |

## 3. 主要機能

### 3.1 欲しいものリスト機能（子ども側）

* 子どもがいつでもお願いごとをリストに追加できる
* **最大5個まで**（満杯のときは何か消さないと追加できない）
* 欲しい度を★1〜3でつけられる
* 子どもの「取捨選択」「優先順位づけ」「待つ経験」を自然に促す

### 3.2 欲しいものリスト承認機能（親側）

* 子どものリストを確認し、今日叶えられるものに承認を出す
* 欲しい度が高いものを優先して叶えやすい
* 承認されたお願いごとが「今日の宝」として冒険にセットされる
* 誕生日・クリスマスのプレゼント選びの参考にもなる

### 3.3 宝セット機能（親用）

* **地図あり**：親が公園の地図上でボールの場所を指定して配置
* 探索中のレーダー画面とは異なり、セット時のみ地図を使用
* ボールごとにクエストを割り当てられる（任意）
  * クエスト例：宿題を終わらせる・草むしりを手伝う・部屋を片付ける・早起きする
  * クエストなしでもOK（2〜3歳はボール発見だけで十分）
* 難易度はボール数で選択可能

| モード | ボール数 | クエスト | 対象年齢 |
| --- | --- | --- | --- |
| かんたん | 3個 | なし推奨 | 2〜3歳 |
| ふつう | 5個 | なし or あり | 4〜5歳 |
| フルクエスト | 7個 | あり推奨 | 6歳〜 |

### 3.4 レーダー画面（メイン）

* ドラゴンレーダー風の円形UI（地図なし・方向と距離だけ表示）
* 地図を表示しないことで「体を動かして探す」体験を最優先にする
* 原作同様、どの方向に・どのくらい近いかだけがわかる
* 近づくほど点滅が速くなる
* 近づくほど音が高くなる・テンポが上がる
* GPS連動でリアルタイム更新
* 半径5メートル以内に近づくと発見判定
* 残り何個かを常に表示
* **ズーム機能（手動・3段階）**：ボタンで切り替え

| ズーム | 表示範囲 | 使いどころ |
| --- | --- | --- |
| 広域 | 半径100m | 最初に大まかな方向をつかむ |
| 中域 | 半径30m | ある程度絞れてきたとき |
| 近接 | 半径10m | もうすぐ！というとき |

* **距離目盛り**：同心円に距離ラベル（例：広域なら50m・100m、近接なら5m・10m）を表示し、ボールまでの距離感を視覚的に把握できる

### 3.5 発見演出

* 宝箱が開くアニメーション
* クエストが設定されている場合は「このボールのクエストは…〇〇！」と表示
* クエストなしの場合はそのまま発見カウント
* 残り個数カウントダウン
* ファンファーレ・エフェクト

### 3.6 クエストフェーズ（帰宅後・クエストありの場合）

* 集めたボールのクエスト一覧が表示される
* 子どもが1つずつクリアしていく
* 親が確認・承認ボタンを押してクリア判定
* クエストなしのボールはスキップ

### 3.7 神龍召喚（全ボール発見＋全クエストクリア後）

* 全条件達成で神龍召喚演出
* 承認済みのお願いごとが最大3つ表示され、子どもが1つ選ぶ
* 選んだお願いごとが「叶えたものアルバム」に記録される

### 3.8 叶えたものアルバム

* 神龍召喚で叶えたお願いごとが自動的にアルバムに記録される
* 叶えた瞬間に写真を撮って紐付けられる
* カード形式で「叶えたお願いごと＋日付＋写真」が並ぶ
* 数年後に見返すと家族の成長記録・思い出になる
* 叶えたものはリストから消えて枠が空く（新しい夢を入れる余白に）

### 3.9 設定機能

ユーザーが好みに合わせてカスタマイズできる設定画面。

| 設定項目 | 選択肢 | 備考 |
| --- | --- | --- |
| レーダースタイル | ⭕ 円 / ⊞ 方眼 / 🔀 切替 / ✨ 重ね | デフォルト：方眼 |
| ボール数（難易度） | 3個 / 5個 / 7個 | デフォルト：7個 |
| 親モード認証 | WebAuthn（指紋・顔認証）/ PINコード（4桁） | デフォルト：PIN |

* **クエスト** — あり / なし
* **親モード認証** — 子どもが勝手に親モードに切り替えられないようにする
  * WebAuthn：iPhoneのFace ID / Touch ID、AndroidのAFP認証をブラウザから利用
  * PINコード：親が設定した4桁のPINを入力して切り替え（フォールバック）
  * 本実装（Claude Code）時に組み込む
* **ボール表示スタイル** — 見やすさの好みに合わせて選択可能
  * 大きく太く：数字を大きく極太で表示
  * ボール大きめ：ボール自体を大きくして文字スペースを確保
  * ★で表現：★1〜3は星マーク、4〜7は数字＋★で直感的に
  * 黒縁取り：数字に黒い縁取りをつけてコントラスト強調
* **⊞ 方眼** — ドラゴンレーダー感・世界観重視
* **🔀 切替** — 広域・中域は方眼、近接になると自動で円に切り替わる
* **✨ 重ね** — 方眼（薄）＋円（濃）を同時表示・両立タイプ

### 3.10 コレクション機能

* 集めた冒険の記録（日付・公園名・達成したお願いごと）
* 「次はどんなお願いが叶うかな」という来訪動機に

## 4. 非機能要件

| 項目 | 内容 |
| --- | --- |
| 対応環境 | スマホブラウザ（Web）優先 |
| GPS精度 | リアルタイム位置情報取得・半径5メートル以内で発見判定 |
| 音 | 距離に応じた音のフィードバック |
| UI | 子どもが見ても楽しいシンプルなビジュアル |
| オフライン | 宝セット用地図タイルのキャッシュによるオフライン対応 |
| データ永続化 | 欲しいものリスト・叶えたものアルバムはDBに保存（セッションをまたいで参照可能） |
| 地図方針 | 宝セット画面のみ地図あり・レーダー画面は地図なし（方向と距離のみ） |

## 5. 画面構成（概要）

```
├── ホーム画面
│   ├── 冒険スタート
│   └── 欲しいものリスト（子ども・親共有）
├── 欲しいものリスト画面
│   ├── 子ども：お願いごと登録（最大5個・欲しい度★）
│   └── 親：承認・今日の宝に設定
├── 宝セット画面（親用）
│   ├── 地図上で宝を配置
│   └── 難易度選択（3・5・7個モード）
├── レーダー画面（メイン）
│   ├── ドラゴンレーダー風UI
│   ├── 距離・方向・残り個数表示
│   └── 音・点滅フィードバック
├── 発見画面
│   └── 宝箱オープンアニメーション
├── クエスト画面（帰宅後・クエストありの場合）
│   ├── クエスト一覧（ボール別）
│   └── 親の承認ボタン
├── 神龍召喚画面（全条件達成時）
│   ├── 神龍召喚演出
│   └── お願いごと選択（最大3つから1つ）
├── 叶えたものアルバム画面
│   ├── 叶えたお願いごとカード（日付・写真）
│   └── 家族の思い出一覧
├── コレクション画面
│   └── 冒険履歴（日付・公園・達成お願いごと）
├── 設定画面
│   ├── レーダースタイル選択（円・方眼・切替・重ね）
│   ├── ボール数（難易度）選択（3・5・7個）
│   ├── クエストあり / なし
│   └── プロフィール・ログイン（Phase 2）
```
