## 2. 機能要件

### 2.1 欲しいものリスト（子ども側）

* お願いごとを自由入力で登録できる
* 最大5個まで（満杯のときは既存を削除しないと追加できない）
* 欲しい度を★1〜3で設定できる
* 一覧表示で欲しい度順に並べ替えられる
* 承認済み・未承認のステータスが表示される

### 2.2 欲しいものリスト（親側）

* 子どものリストを確認できる
* 1つ以上のお願いごとに承認を出せる（今日の冒険で叶える対象）
* 承認したものが宝セット画面に引き継がれる
* 承認を取り消せる

### 2.3 宝セット機能

* OpenStreetMapで公園の地図を表示できる
* 地図をタップしてボールを配置できる
* ボール数を3・5・7個から選択できる
* ボールごとにクエストを設定できる（プリセット選択 or 自由入力）
* クエストなし設定も可能
* 配置したボールを削除・移動できる
* 設定完了後、冒険を開始できる

### 2.4 レーダー画面

* ドラゴンレーダー風の円形UIを表示する
* Geolocation APIでリアルタイムの位置情報を取得する
* 各ボールの方向をレーダー上のドットで示す
* 各ボールまでの距離に応じてドットの位置を更新する
* ズームを3段階で切り替えられる（広域100m / 中域30m / 近接10m）
* ズームはボタン1つで順番に切り替わる
* 近づくほど点滅が速くなる
* 距離目盛りを表示する（設定したスタイルで）
* 残り発見数を常に表示する
* 半径5メートル以内に近づくと発見判定になる

### 2.5 発見演出

* ボール発見時にアニメーションを表示する
* クエストが設定されている場合はクエスト内容を表示する
* 残り個数を表示する
* 「つぎをさがす」「クエストをみる」を選択できる

### 2.6 クエスト管理

* 発見したボール分のクエストがリストに表示される（発見と同時に追加）
* 子どもがやったチェックをつけられる（👧マーク）
* 親がちゃんとできたか確認して承認できる（👨マーク）
* 子どもチェック前は親承認できない（2段階制御）
* クエストなしのボールはスキップされる
* 全クエスト親承認済みになると次のフェーズに進める

### 2.7 シェンロン召喚フロー

* 全ボール発見＋全クエスト親承認でシェンロン召喚演出を表示する
* 親が承認したお願いごとを最大3つ表示する
* 子どもが1つ選ぶ
* 選んだお願いごとを親が「かなえてあげる」で承認する
* 承認後、子どもがアルバムへの一言を入力できる
* 入力後にアルバムに記録される

### 2.8 叶えたものアルバム

* 叶えたお願いごと・日付・一言が記録される
* カード形式で一覧表示される
* 写真を撮って紐付けられる（カメラアクセス）
* DBに永続化され、次回以降も閲覧できる

### 2.9 設定機能

* レーダースタイルを4種から選択できる（⭕円 / ⊞方眼 / 🔀切替 / ✨重ね）
* ボール表示スタイルを4種から選択できる（大きく太く / ボール大きめ / ★で表現 / 黒縁取り）
* ボール数（難易度）を3・5・7個から選択できる
* クエストあり / なしを切り替えられる
* 親モード認証方式を設定できる（WebAuthn / PINコード4桁）

### 2.10 親モード認証

* こどもモード↔親モードの切り替え時に認証を求める
* WebAuthn（Face ID / Touch ID / Android指紋認証）をブラウザから利用する
* WebAuthn未対応の場合はPINコード（4桁）にフォールバックする
* 親がPINを設定・変更できる

## 3. 非機能要件

| 項目 | 内容 |
| --- | --- |
| 対応環境 | スマホブラウザ（iOS Safari / Android Chrome）優先 |
| GPS精度 | Geolocation API・高精度モード（enableHighAccuracy: true） |
| 発見判定 | 半径5メートル以内に近づいたとき |
| レーダー更新間隔 | 1秒以内 |
| 音 | Web Audio APIで距離に応じた音のフィードバック |
| 地図 | 宝セット画面のみOpenStreetMap（Leaflet.js）を使用 |
| オフライン | 宝セット用地図タイルをキャッシュ（Service Worker） |
| データ永続化 | 欲しいものリスト・アルバムはDB保存（セッションをまたいで参照可能） |
| レーダー・クエスト | セッション内メモリで管理（DBなし） |
| パフォーマンス | レーダー画面の表示・更新は2秒以内 |

## 4. 画面一覧

| 画面名 | 概要 | 主なユーザー |
| --- | --- | --- |
| ホーム画面 | 冒険スタート・欲しいものリストへのナビ | 親・子ども |

| シェンロン画面 | 召喚演出・お願いごと選択・親承認・一言入力（タブ：シェンロン） | 親・子ども |
| アルバム画面 | 叶えたお願いごとの一覧・写真 | 親・子ども |
| 設定画面 | レーダー・ボール・認証の各種設定（タブ：設定） | 親 |

## 5. 対象外（Phase 1では作らない）

* ユーザー登録・ログイン認証（JWT）
* マルチデバイス対応（データ同期）
* 他の親子が設置した宝を探すコミュニティ機能
* 子どもが単独でレーダーを操作するモード
* 公園ごとのスタンプラリー
* AR機能
* アルバムのPDF・写真ブックエクスポート
* 季節・イベント限定の特別アイテム
