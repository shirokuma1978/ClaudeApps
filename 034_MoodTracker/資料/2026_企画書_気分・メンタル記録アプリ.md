# 気分・メンタル記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

メンタルヘルスへの関心が高まる中、自分の感情・気分の変化を記録・理解することが重要である。気分・メンタル記録アプリは以下の課題を解決する。

- 毎日の気分・感情を記録して自己理解を深める
- 気分の変化パターンを分析して、何が自分に影響するか把握する
- メンタル不調の早期発見と、精神科・心療内科への情報提供に役立てる

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | カウンセラー・医師とのデータ共有 |

---

## 3. 主要機能

### 3.1 気分記録機能（MVP）
- 気分の記録（複数回/日）
- 記録項目：
  - 日時
  - 気分スコア（1〜10段階）
  - 感情タグ（嬉しい・悲しい・不安・疲れた等）
  - エネルギーレベル
  - 気分の原因・きっかけ
  - 自由メモ

### 3.2 感情ログのタグ管理
- よく使う感情タグのカスタマイズ
- タグ別の発生頻度分析

### 3.3 トリガー分析
- 特定のイベント・状況との気分の関係
- 睡眠・運動との相関分析

### 3.4 グラフ・可視化
- 気分スコアの時系列グラフ
- 週間・月間の感情分布
- 最良日・最悪日の振り返り

### 3.5 医療連携（Phase 3）
- 気分記録のPDFレポート生成
- カウンセラーへの共有機能

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| プライバシー | 最高レベルのデータ保護 |
| データ保存 | ユーザーごとのデータ暗号化保存 |
| パフォーマンス | 記録は1秒以内 |
| UI | シンプルで感情移入を促さないデザイン |

