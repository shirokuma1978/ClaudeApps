# カロリー計算アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

カロリー管理はダイエット・体重管理の基本であるが、毎食の計算は手間がかかる。カロリー計算アプリは以下の課題を解決する。

- 毎食のカロリーを素早く記録し、1日の総カロリーを把握する
- 目標カロリーとの差分をリアルタイムで確認できる
- 消費カロリーとの収支バランスで体重変化を予測する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | ダイエット仲間との進捗共有 |

---

## 3. 主要機能

### 3.1 食事記録・カロリー計算（MVP）
- 食事の記録
- 食品データベースからの選択
- 量の調整（グラム・個数）
- 自動カロリー計算
- 1日の合計表示

### 3.2 食品データベース
- 日本食品標準成分表ベース
- コンビニ・外食チェーンの定番メニュー
- カスタム食品の登録

### 3.3 目標設定・トラッキング
- 目標カロリー設定（TDEE計算）
- 残りカロリーのリアルタイム表示
- 週間・月間の推移グラフ

### 3.4 消費カロリー管理
- 運動記録との連携
- 基礎代謝・活動量の設定
- 摂取/消費カロリーバランス表示

### 3.5 体重変化予測（Phase 3）
- カロリー収支から体重変化を予測
- 目標体重達成日の計算
- 週次レポートの生成

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |
| 食品DB | 日本食品標準成分表のCSV取込 |
| データ保存 | ユーザーごとのデータ永続化 |
| 検索速度 | 食品検索は2秒以内 |
| UI | 素早い入力のためのシンプルデザイン |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今日のカロリーサマリー・残りカロリー）

├── 食品検索画面
├── 今日の食事一覧
├── 週間・月間グラフ画面
├── 目標設定画面
└── 設定画面（身体情報・活動量設定）
```
