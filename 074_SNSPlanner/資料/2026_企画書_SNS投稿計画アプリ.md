# SNS投稿計画アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

個人・ビジネスでのSNS運用において、コンテンツの計画・管理・分析が課題になる。SNS投稿計画アプリは以下の課題を解決する。

- 複数SNSの投稿コンテンツを一元管理する
- 投稿スケジュールをカレンダーで可視化して継続的な発信を支援する
- 過去の投稿・反応を記録して次のコンテンツ戦略に活かす

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人SNS運用者） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 企業SNS担当チームでの共同管理 |

---

## 3. 主要機能

### 3.1 投稿管理機能（MVP）
- 投稿の作成・編集・削除
- 登録項目：
  - 投稿テキスト
  - 対象SNS（Instagram・Twitter・TikTok等）
  - 予定投稿日時
  - ステータス（下書き・承認待ち・投稿済み）
  - 画像・動画添付
  - ハッシュタグ
  - 投稿後の反応（いいね・コメント数等）

### 3.2 コンテンツカレンダー
- 月間・週間の投稿スケジュール表示
- SNS別のカラーコーディング
- ドラッグ&ドロップでのスケジュール変更

### 3.3 コンテンツアイデア管理
- 投稿ネタのストック
- テーマ・シリーズの管理
- コンテンツタイプ別（写真・動画・ストーリー等）の分類

### 3.4 パフォーマンス記録（Phase 2）
- 投稿後のエンゲージメント記録
- フォロワー数の推移
- 最もパフォーマンスの良いコンテンツタイプの分析

### 3.5 チーム管理（Phase 3）
- 複数担当者での投稿管理
- 承認ワークフロー

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| カレンダー | インタラクティブなコンテンツカレンダー |
| データ保存 | ユーザーごとのデータ永続化 |
| 画像 | 投稿画像のプレビュー・管理 |
| パフォーマンス | カレンダー表示は3秒以内 |

---

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今週の投稿予定・下書き一覧）
├── コンテンツカレンダー画面
├── 投稿作成・編集画面
├── コンテンツアイデア帳
├── パフォーマンス分析画面
└── 設定画面（SNSアカウント・カテゴリ設定）
```
