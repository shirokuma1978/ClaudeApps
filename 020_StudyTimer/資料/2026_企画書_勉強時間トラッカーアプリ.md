# 勉強時間トラッカーアプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

勉強の生産性を高めるには、集中時間の計測と管理が重要である。勉強時間トラッカーアプリは以下の課題を解決する。

- ポモドーロ・テクニック等を活用した集中勉強を支援する
- 科目・テーマごとの勉強時間を記録して偏りを把握する
- 目標勉強時間の設定と達成状況の可視化でモチベーションを維持する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 受験仲間・勉強仲間との進捗共有 |

---

## 3. 主要機能

### 3.1 タイマー機能（MVP）
- ポモドーロタイマー（25分集中・5分休憩）
- カスタムタイマー設定
- バックグラウンドでの動作
- セッション完了通知

### 3.2 学習セッション記録
- 勉強開始・終了の記録
- 科目・テーマの選択
- セッションごとのメモ
- 中断・休憩の記録

### 3.3 目標・統計管理
- 日次・週次の勉強時間目標設定
- 科目別の時間配分グラフ
- 月間・年間の勉強時間推移
- 連続勉強日数（ストリーク）

### 3.4 集中力分析（Phase 2）
- 時間帯別の集中度記録
- 最も集中できる時間帯の分析
- 休憩最適化の提案

### 3.5 勉強仲間機能（Phase 3）
- 勉強時間のリアルタイム共有
- 一緒に勉強する「バーチャル自習室」

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| バックグラウンド | タブを閉じても継続するタイマー |
| 通知 | ブラウザ通知でポモドーロ終了を知らせる |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | タイマー精度は1秒以内の誤差 |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（タイマー画面・今日の進捗）

├── 勉強記録一覧画面
├── 統計・グラフ画面
├── 目標設定画面
└── 設定画面（タイマー設定・通知設定）
```
