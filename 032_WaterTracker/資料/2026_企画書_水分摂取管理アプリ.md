# 水分摂取管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

適切な水分摂取は健康維持の基本だが、意識しないと不足しがちである。水分摂取管理アプリは以下の課題を解決する。

- 1日の水分摂取量を記録・管理して目標達成をサポートする
- 定期的なリマインダーで水を飲む習慣を形成する
- 水分不足による体調不良を予防する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族の水分管理（子供・高齢者） |

---

## 3. 主要機能

### 3.1 水分記録機能（MVP）
- 水分摂取のワンタップ記録
- 飲み物の種類と量の記録
  - 水・お茶・コーヒー・スポーツ飲料等
  - カスタム量設定（200ml・500ml等）
- 1日の合計摂取量表示
- 目標達成率のリングゲージ表示

### 3.2 目標設定機能
- 体重・運動量に基づく推奨水分量の計算
- 日別・季節別の目標調整

### 3.3 リマインダー機能（Phase 2）
- 定期リマインダー（例：1時間ごと）
- カスタムリマインダー時間設定

### 3.4 統計・グラフ
- 週間・月間の水分摂取量グラフ
- 時間帯別の摂取パターン
- 目標達成率の推移

### 3.5 健康連携（Phase 3）
- 体重・睡眠との相関分析
- ヘルスデータとの統合

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホブラウザ（Web）優先 |
| UI | ワンタップで素早く記録できるシンプルUI |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 記録は1秒以内 |
| 通知 | ブラウザ通知によるリマインダー |

## 5. 画面構成（概要）

```
├── ログイン画面
├── ユーザー登録画面
├── ホーム（今日の摂取量・達成ゲージ）

├── 今日の記録一覧
├── 週間・月間グラフ画面
├── 目標設定画面
└── 設定画面（リマインダー・飲み物プリセット）
```
