# 家計簿アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

お金の流れを可視化することは、無駄な出費を減らし、貯蓄目標を達成するための第一歩である。家計簿アプリは以下の課題を解決する。

- 収入・支出を記録して家計の現状を把握する
- カテゴリ別の支出傾向を分析し、節約ポイントを見つける
- 月次・年次の家計レポートで長期的な財務状況を管理する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 夫婦・家族での家計共有管理 |

---

## 3. 主要機能

### 3.1 収支記録機能（MVP）
- 収入・支出の登録・編集・削除
- 登録項目：
  - 日付
  - 金額
  - 種別（収入・支出）
  - カテゴリ（食費・交通費・娯楽費など）
  - 支払い方法（現金・カード・電子マネー）
  - メモ

### 3.2 予算管理機能
- カテゴリ別の月次予算設定
- 予算に対する消化率の表示
- 予算超過アラート

### 3.3 集計・グラフ機能
- 月別の収支サマリー
- カテゴリ別支出の円グラフ
- 収支推移の折れ線グラフ
- 前月・前年比較

### 3.4 固定費管理
- 家賃・サブスクなど固定費の登録
- 毎月の自動計上

### 3.5 レポート機能（Phase 3）
- 月次・年次レポートの自動生成
- CSVエクスポート
- 節約アドバイスの表示

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| レスポンシブ | スマホでの素早い入力を優先 |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | 入力・集計は3秒以内 |
| セキュリティ | 金融データの適切な保護 |
