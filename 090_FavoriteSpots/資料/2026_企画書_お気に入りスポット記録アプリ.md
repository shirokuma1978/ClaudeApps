# お気に入りスポット記録アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

「あの公園の紅葉が最高だった」「あの展望台からの夜景は忘れられない」という思い出のスポットを記録・共有したいというニーズがある。お気に入りスポット記録アプリは以下の課題を解決する。

- 訪れて感動したスポットを記録してまた行けるようにする
- ジャンル別・シーン別のお気に入りスポットを整理する
- 大切な人に「ここに連れて行きたい」スポットを共有する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（お出かけ好き） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 友人・家族とのスポット共有 |

---

## 3. 主要機能

### 3.1 スポット登録機能（MVP）
- お気に入りスポットの登録・編集・削除
- 登録項目：
  - スポット名
  - 場所（住所・GPS座標）
  - カテゴリ（自然・グルメ・景色・歴史等）
  - シーン（デート・家族・一人旅等）
  - 季節（春夏秋冬）
  - 訪問日
  - 評価・おすすめ度
  - 感想・ポイント
  - 写真

### 3.2 マップ表示
- 登録スポットの地図表示
- エリア別の絞り込み
- 「今いる場所の近く」のスポット表示

### 3.3 シーン別・季節別管理
- デートスポット・家族お出かけスポット等の分類
- 季節ごとのおすすめスポット
- 「この時期に行くべき場所」

### 3.4 ウィッシュリスト
- まだ行っていないが行きたいスポット
- 実際に行ったら記録に変換

### 3.5 友人共有（Phase 3）
- スポットのURL共有
- 「一緒に行こう」リスト

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応 |
| GPS | 現在地からの近くのスポット表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| 地図 | Google Maps / Leaflet.js |
| 画像 | スポット写真のアップロード |

```
├── 地図ビュー画面
├── シーン別・季節別一覧
├── ウィッシュリスト画面

```
