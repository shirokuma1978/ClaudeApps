# レシピ管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

料理のレシピは散在しがちで、いざ作ろうとするときに「あのレシピどこだっけ？」となることが多い。レシピ管理アプリは以下の課題を解決する。

- 自分のレシピを一元管理し、いつでも参照できる
- 材料からレシピを検索し、献立決めを効率化する
- 作った料理の写真と評価を記録し、改良の記録を残す

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（個人利用） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | 家族・友人とのレシピ共有 |

---

## 3. 主要機能

### 3.1 レシピ登録機能（MVP）
- レシピの作成・編集・削除
- 登録項目：
  - 料理名
  - カテゴリ（和食・洋食・中華など）
  - 材料・分量（人数調整対応）
  - 手順（ステップ形式）
  - 調理時間
  - 難易度
  - タグ
  - メモ・ポイント

### 3.2 レシピ検索機能
- 料理名・材料名での検索
- カテゴリ・タグでのフィルタリング
- 調理時間・難易度での絞り込み

### 3.3 調理記録機能
- 作った日の記録
- 写真のアップロード
- 評価（5段階）・改良メモ

### 3.4 材料リスト連携
- レシピから買い物リストを自動生成
- 複数レシピの材料をまとめて買い物リストへ

### 3.5 URL取り込み機能（Phase 2）
- 外部レシピサイトのURL からレシピを取り込み
- クックパッド・レシピブログ等の対応

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | スマホ・PCブラウザ対応（調理中のスマホ利用を考慮） |
| レスポンシブ | 調理中でも見やすい大文字表示モード |
| データ保存 | ユーザーごとのデータ永続化 |
| パフォーマンス | レシピ検索は3秒以内 |
| 画像 | 料理写真のアップロード・表示 |

```
├── レシピ詳細画面（調理中モード）
├── レシピ登録・編集画面
├── 調理記録画面
├── 買い物リスト連携画面
└── 設定画面
```
