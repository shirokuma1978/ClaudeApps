# キャリア目標管理アプリ 企画書

**バージョン:** 1.0
**作成日:** 2026年3月27日
**ステータス:** 企画中

---

## 1. 背景・目的

キャリアアップを目指すビジネスパーソンにとって、長期的な目標設定と進捗管理が成功の鍵となる。キャリア目標管理アプリは以下の課題を解決する。

- キャリアの長期目標・短期目標を体系的に管理する
- 目標達成に向けたアクションプランを立案・実行する
- キャリアの成長を可視化してモチベーションを維持する

---

## 2. ターゲットユーザー

| フェーズ | ユーザー |
|---------|---------|
| Phase 1（初期） | 自分自身（キャリアアップを目指すビジネスパーソン） |
| Phase 2（認証） | ユーザー登録・ログインによるマルチユーザー対応 |
| Phase 3（拡張） | キャリアコーチ・メンターとの目標共有 |

---

## 3. 主要機能

### 3.1 キャリア目標登録機能（MVP）
- 目標の登録・編集・削除
- 登録項目：
  - 目標タイトル
  - 目標カテゴリ（スキル・役職・年収・転職・独立等）
  - 時間軸（1年・3年・5年・10年）
  - 具体的な目標内容
  - 現在地（現状の把握）
  - 達成指標（KPI）
  - 期限
  - 優先度
  - ステータス（未着手・進行中・達成・見直し）

### 3.2 アクションプラン管理
- 目標達成のためのステップ設定
- マイルストーンの管理
- 週次・月次のアクション登録
- 進捗率の更新

### 3.3 キャリアタイムライン
- 過去の経歴・実績の記録
- 目標達成履歴の可視化
- キャリアの流れをタイムラインで表示

### 3.4 振り返り・レビュー機能（Phase 2）
- 月次・四半期の振り返り記録
- 目標の修正・更新履歴
- 成長の記録とナレッジ蓄積

### 3.5 メンター連携（Phase 3）
- コーチ・メンターへの目標共有
- フィードバックの記録
- 定期的なチェックインの管理

---

## 4. 非機能要件

| 項目 | 内容 |
|-----|-----|
| 対応環境 | PC・スマホブラウザ対応 |
| 可視化 | 目標進捗のグラフ・チャート表示 |
| データ保存 | ユーザーごとのデータ永続化 |
| リマインド | 目標期限・振り返りのリマインダー |
| パフォーマンス | 一覧表示は3秒以内 |
